关闭 Linux bash 中使用 tab 以及 vim 中操作错误的提示音。

### bash
修改文件 `/etc/inputrc`，加入
```bash
set bell-style none
```

### vim
修改文件 `/etc/vim/vimrc`，加入
```vi
:set vb t_vb=
```
