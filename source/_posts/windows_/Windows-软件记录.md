* [Capslock+](https://capslox.com/capslock-plus/)
  * 博客[快捷使用方向键](https://zhuxz0299.github.io/posts/aae19956.html)中有记录。
* [im-select](https://github.com/daipeihust/im-select)
  * 为了方便在使用 VSCodeVim 插件时，进入 NORMAL 模式时能够自动切换英文输入法的辅助工具。
* [iperf](https://iperf.fr/)
  * 一个用于测量两台机器之间带宽与延迟的工具。
* [Openlist](https://github.com/OpenListTeam/OpenList)
  * 博客 [WebDAV 协议与网盘的本地挂载](https://zhuxz0299.github.io/posts/85c3534f.html)中有记录
* [SumatraPDF](https://www.sumatrapdfreader.org/free-pdf-reader)
  * 一个 PDF 阅读器，足够轻量且支持对文档进行简单的反色。
* [TboxWebdav](https://github.com/1357310795/TboxWebdav)
  * 让用户可以通过 WebDAV 协议访问新交大云盘。
* [v2rayN](https://github.com/2dust/v2rayN)
  * 一个图形化的代理工具，支持 Xray，sing-box 等内核。
* [ventory](https://www.ventoy.net/cn/index.html)
  * 制作可启动U盘的开源工具，支持各种操作系统。


* [7-zip](https://www.7-zip.org/)
  * 非常好用的压缩/解压缩工具。
* [Adobe Premiere Pro](https://www.adobe.com/products/premiere.html)
  * 专业的视频剪辑软件。
* [AMD Software](https://www.amd.com/)
  * AMD 显卡驱动与管理软件。
* [Apollo](https://github.com/ClassicOldSong/Apollo)
  * 远程串流软件，是 Sunshine 的 Fork，增加了更多的高级功能。
* Application Verifier
  * 微软官方的应用 BUG 监测工具。不知道哪来的，可能是下载 Visual Studio 附带的。
* [AutoHotkey](https://www.autohotkey.com/)
  * 在 Windows 上用来给各种组合键自定义脚本，是非常好用的工具。
* [Autorun Organizer 下载地址（找不到官网）](https://www.chemtable.com/autorun-organizer.htm)
  * 用于设置 Windows 的开机自启动项，可以禁用某些自启动，也可以设置开机之后一定时间再自启动，缓解刚开机是大量应用同时启动导致的卡顿。
* [BetterGI](https://bettergi.com/)
  * 原神脚本软件。
* [Cherry Studio](https://www.cherry-ai.com/)
  * 提供和 AI 交互的 GUI。
* [Clash Verge](https://github.com/clash-verge-rev/clash-verge-rev)
  * 非常强大的代理软件。
* [DAUM PotPlayer](https://potplayer.tv/)
  * 开箱即用且功能丰富的视频播放器。
* [Docker-desktop](https://www.docker.com/products/docker-desktop/)
  * 目前想在 Windows 上用 Docker 不得不品的一环。
* dotnet
  * .NET 开发环境
* [eudic](https://www.eudic.net/v4/en/app/eudic)
  * 欧陆词典，目前用下来比较好用且美观的本地词典。
* [Git](https://git-scm.com/)
  * git 工具
* [Google Chrome](https://www.google.com/chrome/)
  * Chrome 浏览器 
* [HandBrake](https://handbrake.fr/)
  * 开源视频格式转换工具
* [HP Print and Scan Doctor](https://support.hp.com/us-en/help/printscandoctor)
  * 惠普打印机诊断工具。
* [JetBrains](https://www.jetbrains.com/)
  * JetBrains 全家桶，强大的 IDE 集合。
* [LocalSend](https://localsend.org/)
  * 开源跨平台局域网文件传输工具。
* [Logitech G Hub](https://www.logitechg.com/zh-cn/innovation/g-hub.html)
  * 罗技游戏外设管理驱动。
* [Logi Options+](https://www.logitech.com/zh-cn/software/logi-options-plus.html)
  * 罗技办公外设管理驱动。
* [Mechrevo Control Center](https://www.mechrevo.com/)
  * 机械革命电脑控制中心。
* [Microsoft Office](https://www.office.com/)
  * 微软办公套件。
* [Microsoft OfficePLUS](https://www.officeplus.cn/)
  * 微软官方的 Office 插件，提供模板和素材。
* [Microsoft OneDrive](https://onedrive.live.com/about/zh-cn/)
  * 微软推出的云存储服务。
* [Microsoft SQL Server](https://www.microsoft.com/zh-cn/sql-server)
  * 微软推出的关系型数据库管理系统。
* [Microsoft Visual Studio](https://visualstudio.microsoft.com/)
  * 微软推出的集成开发环境（IDE）。
* [MiHoYo Launcher](https://www.mihoyo.com/)
  * 米哈游启动器，用于启动原神、崩坏：星穹铁道等游戏。
* [Moonlight](https://moonlight-stream.org/)
  * 基于 NVIDIA GameStream 的开源游戏串流客户端。
* [Mozilla Firefox](https://www.mozilla.org/zh-CN/firefox/new/)
  * 优秀的开源浏览器。
* [Mozilla Thunderbird](https://www.thunderbird.net/)
  * Mozilla 出品的开源邮件客户端。
* [Node.js](https://nodejs.org/)
  * JavaScript 运行时环境。
* [OBS Studio](https://obsproject.com/)
  * 强大的开源录屏与直播软件。
* [OPPO Connect](https://connect.oppo.com/zh-CN)
  * OPPO 跨屏互联，手机与电脑协同工具。
* [PowerToys](https://github.com/microsoft/PowerToys)
  * 微软官方出品的系统增强工具集，非常实用。
* [qBittorrent](https://www.qbittorrent.org/)
  * 开源 BitTorrent 客户端。
* [RaiDrive](https://www.raidrive.com/)
  * 将网盘挂载为本地磁盘的工具。
* [Sangfor EasyConnect](https://www.sangfor.com.cn/)
  * 深信服 EasyConnect，用于远程接入办公网络。在电脑上实际上大概是交大 VPN 用到的 64 位的库。
* [SimpleTex](https://simpletex.cn/)
  * 强大的数学公式识别截图 OCR 工具，不过目前来看不如直接用 AI 识别了。
* [Tailscale](https://tailscale.com/)
  * 基于 WireGuard 的零配置虚拟局域网组网工具。
* [Tencent QQ](https://im.qq.com/)
  * 腾讯即时通讯软件。
* [WeChat](https://weixin.qq.com/)
  * 微信电脑版。
* [Tencent Meeting](https://meeting.tencent.com/)
  * 腾讯会议。
* [ToDesk](https://www.todesk.com/)
  * 远程桌面控制软件。
* [Typora](https://typora.io/)
  * 优雅的 Markdown 编辑器（所见即所得）。
* [ViGEmBus](https://github.com/nefarius/ViGEmBus)
  * 虚拟游戏手柄驱动，常配合手柄映射工具使用。不知道哪来的，可能跟着 Apollo 一起装的？
* [Zotero](https://www.zotero.org/)
  * 开源文献管理工具。
