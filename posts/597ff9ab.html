set

通常使用自平衡二叉搜索树（例如红黑树）来实现

初始化

方法

  • .begin() 返回 vector 中第一个元素的 iterator
  • .end() 返回 vector 中最后一个元素后面的 iterator
  • .size() 返回 vector 大小
  • .max_size() 返回目前最多能装多少元素
  • .empty() 是否为空
  • .insert(g) 将元素 ggg 插入集合
  • .erase(g) 将元素 ggg 从集合中移除
  • .find(g) 返回元素 ggg 对应的 iterator
  • .count(g) 根据是否存在对应元素返回 111 或 000
  • .lower_bound(g) 找到第一个不小于 ggg 的元素的 iterator
  • .upper_bound(g) 找到第一个大于 ggg 的元素的 iterator

时间复杂度

插入、删除、查找: O(log⁡N)O(\log N)O(logN)
创建或清除: O(N)O(N)O(N)

multiset

同样基于红黑树实现，和 set 基本相同，仅仅是允许有重复元素。

方法

基本与 set 相同

  • .erase(g) 将所有元素 ggg 从集合中移除
  • .erase(it) 将 iterator it 指向的元素移除
  • .count(g) 返回元素 ggg 的数量
  • .find(g) 返回第一个元素 ggg 对应的 iterator
  • .equal_range(g) 返回一对 iterator，分别指向第一个等于 5 的元素和第一个大于 5 的元素。
