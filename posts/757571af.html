一些命令参数解释

  • -n 是 --name 的简写，表示指定名字
  • -f 是 --file 的简写，表示指定环境
  • conda 的子命令 env 是一个模块，用于 管理环境的高级功能，如：
    • 使用 conda env export 导出环境。
    • 使用 conda env create 基于配置文件创建环境。
    • 使用 conda env remove 删除环境。
