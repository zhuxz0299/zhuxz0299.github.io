SAT Solver

Algorithm 1 (Brute force SAT solver). Given a partial truth assignment J\mathcal{J}J, determine whether we can expand it into a total one which satisfies ϕ\phiϕ.
BF(J)BF(\mathcal{J})BF(J):

  • if J\mathcal{J}J is a total assignment and ⟦ϕ⟧J=T\llbracket\phi\rrbracket_{\mathcal{J}} = \mathbf{T}[[ϕ]]J​=T, return SAT
  • if J\mathcal{J}J is a total assignment and ⟦ϕ⟧J=F\llbracket\phi\rrbracket_{\mathcal{J}} = \mathbf{F}[[ϕ]]J​=F, return UNSAT
  • if J\mathcal{J}J is not a total assignment
    • pick a propositional variable ppp which is not assigned a value by J\mathcal{J}J
    • execute BF(J∪[p↦T])BF(\mathcal{J} \cup [p \mapsto \mathbf{T}])BF(J∪[p↦T]) and BF(J∪[p↦F])BF(\mathcal{J} \cup [p \mapsto \mathbf{F}])BF(J∪[p↦F])
    • return SAT if one of the above answers SAT, return UNSAT otherwise

Optimization: A conflict may be detected on a partial assignment!
这种方法就相当于用递归的方式遍历所有可能的 J\mathcal{J}J。

Example:

(p1∨¬p4)∧(¬p2∨p3)∧(p2∨p4)∧(¬p2∨¬p3∨p4)∧(¬p1∨¬p4)(p_{1} \lor \neg p_{4}) \land (\neg p_{2} \lor p_{3}) \land (p_{2} \lor p_{4}) \land (\neg p_{2} \lor \neg p_{3} \lor p_{4}) \land (\neg p_{1} \lor \neg p_{4}) (p1​∨¬p4​)∧(¬p2​∨p3​)∧(p2​∨p4​)∧(¬p2​∨¬p3​∨p4​)∧(¬p1​∨¬p4​)

J=[p1↦T,p4↦T]\mathcal{J} = [p_{1} \mapsto \mathbf{T}, p_{4} \mapsto \mathbf{T}] J=[p1​↦T,p4​↦T]

Algorithm 2. BF+(J)BF^{+}(\mathcal{J})BF+(J):

  • if J\mathcal{J}J causes a conflict on ϕ\phiϕ, return UNSAT
  • otherwise, if J\mathcal{J}J is a total assignment, return SAT
  • otherwise,
    • pick a propositional variable ppp which is not assigned a value by J\mathcal{J}J
    • execute BF+(J∪[p↦T])BF^{+}(\mathcal{J} \cup [p \mapsto \mathbf{T}])BF+(J∪[p↦T]) and BF+(J∪[p↦F])BF^{+}(\mathcal{J} \cup [p \mapsto \mathbf{F}])BF+(J∪[p↦F])
    • return SAT if one of the above answers SAT, return UNSAT otherwise

和上一种方法相比，这种方法能够在 partial truth assignment J\mathcal{J}J UNSAT 的时候退出，减少了枚举数量。

Unit Propagation (赋值推导): In one clause, if all literals except one are false, the last one must be true.
Example:

(p1∨¬p4)∧(¬p2∨p3)∧(p2∨p4)∧(¬p2∨¬p3∨p4)∧(¬p1∨¬p4)J=[p2↦T]⇒J(p3) should be T⇒(p1∨¬p4)∧(¬p2∨p3)∧(p2∨p4)∧(¬p2∨¬p3∨p4)∧(¬p1∨¬p4)J=[p2↦T,p3↦T]⇒J(p4) should be T\begin{aligned} & (p_{1} \vee \neg p_{4}) \wedge \boldsymbol{(\neg p_{2} \vee p_{3})} \wedge (p_{2} \vee p_{4}) \wedge (\neg p_{2} \vee \neg p_{3} \vee p_{4}) \wedge (\neg p_{1} \vee \neg p_{4}) \quad \mathcal{J} = [p_{2} \mapsto \mathbf{T}] \\ \Rightarrow & \quad \mathcal{J}(p_{3}) \text{ should be } \mathbf{T} \\ \Rightarrow & \quad (p_{1} \vee \neg p_{4}) \wedge (\neg p_{2} \vee p_{3}) \wedge (p_{2} \vee p_{4}) \wedge \boldsymbol{(\neg p_{2} \vee \neg p_{3} \vee p_{4})} \wedge (\neg p_{1} \vee \neg p_{4}) \quad \mathcal{J} = [p_{2} \mapsto \mathbf{T}, p_{3} \mapsto \mathbf{T}] \\ \Rightarrow & \quad \mathcal{J}(p_{4}) \text{ should be } \mathbf{T} \end{aligned} ⇒⇒⇒​(p1​∨¬p4​)∧(¬p2​∨p3​)∧(p2​∨p4​)∧(¬p2​∨¬p3​∨p4​)∧(¬p1​∨¬p4​)J=[p2​↦T]J(p3​) should be T(p1​∨¬p4​)∧(¬p2​∨p3​)∧(p2​∨p4​)∧(¬p2​∨¬p3​∨p4​)∧(¬p1​∨¬p4​)J=[p2​↦T,p3​↦T]J(p4​) should be T​

Algorithm 3 (DPLL(J\mathcal{J}J)).

  • let J′\mathcal{J}'J′ be UnitPro(J\mathcal{J}J)
  • if J′\mathcal{J}'J′ causes a conflict on ϕ\phiϕ, return UNSAT
  • otherwise, if J′\mathcal{J}'J′ is a total assignment, return SAT
  • otherwise,
    • pick a propositional variable ppp which is not assigned a value by J′\mathcal{J}'J′
    • execute DPLL(J′∪[p↦T]\mathcal{J}' \cup [p \mapsto \mathbf{T}]J′∪[p↦T]) and DPLL(J′∪[p↦F]\mathcal{J}' \cup [p \mapsto \mathbf{F}]J′∪[p↦F])
    • return SAT if one of the above answers SAT, return UNSAT otherwise

利用了赋值推导的算法，进一步减少了需要枚举的情况。

Algorithm 4 (CDCL, Conflict Driven Clause Learning).
矛盾驱动的子句学习方法。

  • let J\mathcal{J}J be the empty assignment at the beginning
  • set J\mathcal{J}J to UnitPro(J\mathcal{J}J)
  • if J\mathcal{J}J causes a conflict on ϕ\phiϕ
    • let DDD be the result of ConflictClauseGen()
    • if DDD is an empty clause, return UNSAT
    • add DDD to the CNF
    • remove assignments after the second last “Pick” in DDD from J\mathcal{J}J
    • go back to step 2 (the unit propagation step)
  • otherwise, if J\mathcal{J}J is a total assignment, return SAT
  • otherwise,
    • pick a propositional variable ppp which is not assigned a value by J\mathcal{J}J
    • pick a truth value t∈{T,F}t \in \{\mathbf{T}, \mathbf{F}\}t∈{T,F}
    • set J\mathcal{J}J to (J∪[p↦t]\mathcal{J} \cup [p \mapsto t]J∪[p↦t])
    • go back to step 2 (the unit propagation step)
