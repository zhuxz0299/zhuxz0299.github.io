有限域的基本结构与性质

特征与基域

回顾域论基础，对于有限域 FFF：

  • 特征 (Characteristic)：有限域的特征 char(F)=pchar(F) = pchar(F)=p，其中 ppp 为素数。
  • 基域：FFF 必定包含一个同构于 Zp\mathbb{Z}_pZp​ 的素域，这里记为 FpF_{p}Fp​
    • 即 Fp<FF_{p}<FFp​<F，有时候也可以简单的写为 Zp<F\mathbb{Z}_p < FZp​<F
  • 乘法群：有限域的非零元乘法群 (F∗,⋅)(F^*, \cdot)(F∗,⋅) 是循环群。

扩张与阶

设 FFF 是有限域，其代数扩张 E/FE / FE/F 具有以下性质：

  1. 可分性：有限域的代数扩张都是可分的（Separable）。
  2. 单代数扩张：对于某个 α∈E\alpha \in Eα∈E，有 E=F(α)E = F(\alpha)E=F(α)
    • 这一点可以利用 (E∗,⋅)(E^{*}, \cdot )(E∗,⋅) 是循环群证明
  3. 维数与元素个数：
    • 假如 [E:F]=n[E:F] = n[E:F]=n，取 EEE 在 FFF 上的一组基 α1,…αn\alpha_1, \ldots \alpha_nα1​,…αn​
    • EEE 上任意元素可以表示为 ∑i=1naiαi\sum_{i=1}^{n}a_i \alpha_i∑i=1n​ai​αi​，其中 ai∈Fa_i \in Fai​∈F
    • 因此有 ∣E∣=∣F∣n\left\vert E \right\vert = \left\vert F \right\vert^{n}∣E∣=∣F∣n
    • 考虑到上面的 char(F)=p⇒Zp<Fchar(F) = p \Rightarrow \mathbb{Z}_{p}<Fchar(F)=p⇒Zp​<F，那么应当有 ∣F∣=pn\left\vert F \right\vert = p^{n}∣F∣=pn

记号约定：

  • 通常记 q=pnq = p^nq=pn。
  • 有限域记为 FqF_qFq​ 或 GF(q)GF(q)GF(q)，其中 ∣Fq∣=q|F_q| = q∣Fq​∣=q。

有限域都是分裂域

性质
有限域 FqF_qFq​ 可以被刻画为多项式 xq−xx^q - xxq−x 的在素域 Zp\mathbb{Z}_{p}Zp​ 上的分裂域。

根的性质：

  • FqF_qFq​ 中的所有非零元素构成阶为 q−1q-1q−1 的循环群，即 ∀α∈Fq∗,αq−1=1\forall \alpha \in F_q^*, \alpha^{q-1} = 1∀α∈Fq∗​,αq−1=1。
  • 加上零元素，则 ∀α∈Fq,αq=α\forall \alpha \in F_q, \alpha^q = \alpha∀α∈Fq​,αq=α。
  • 这意味着 FqF_qFq​ 的所有元素都是方程 xq−x=0x^q - x = 0xq−x=0 的根。

重根判定：
考虑多项式 f(x)=xq−xf(x) = x^q - xf(x)=xq−x，其导数为：

f′(x)=qxq−1−1=−1≠0(在特征 p 的域中 q=pn=0)f'(x) = qx^{q-1} - 1 = -1 \neq 0 \quad (\text{在特征 } p \text{ 的域中 } q=p^n=0) f′(x)=qxq−1−1=−1=0(在特征 p 的域中 q=pn=0)

由于导数不为 0 且与 f(x)f(x)f(x) 互素，故 xq−xx^q - xxq−x 没有重根。其根的集合大小恰好为 qqq，从而得到根的集合恰好为 FqF_qFq​

利用多项式构造有限域：唯一性

前面我们知道假如 ∣F∣=q\left\vert F \right\vert = q∣F∣=q，那么 ∃n,p\exists n, p∃n,p 使得 q=pnq = p^{n}q=pn。接下来我们考虑：是否对于任意的 nnn，得到的 q=pnq = p^{n}q=pn 都能是某个有限域的大小？事实上，可以通过构造法证明阶为 q=pnq = p^{n}q=pn 的有限域对于 ∀n\forall n∀n 存在且是唯一的。

命题：
考虑 q=pn,n∈N∗q = p^{n}, n \in \mathbb{N}^{*}q=pn,n∈N∗，令 FFF 为多项式 xq−x∈Zp[x]x^q - x \in \mathbb{Z}_{p}[x]xq−x∈Zp​[x] 在其代数闭包中的所有根的集合，则 FFF 构成一个域。

证明：

  1. 减法封闭性：
    对任意 α,β∈F\alpha, \beta \in Fα,β∈F，考察 α−β\alpha - \betaα−β：

    (α−β)q−(α−β)=(αq−βq)−(α−β)=(α−α)−(β−β)=0(\alpha - \beta)^q - (\alpha - \beta) = (\alpha^q - \beta^q) - (\alpha - \beta) = (\alpha - \alpha) - (\beta - \beta) = 0 (α−β)q−(α−β)=(αq−βq)−(α−β)=(α−α)−(β−β)=0

    故 α−β∈F\alpha - \beta \in Fα−β∈F。

  2. 除法封闭性：
    对任意 α∈F,β∈F∖{0}\alpha \in F, \beta \in F \setminus \{0\}α∈F,β∈F∖{0}，考察 αβ−1\alpha \beta^{-1}αβ−1：

    故 αβ−1∈F\alpha \beta^{-1} \in Fαβ−1∈F。

得到 FFF 是一个域，且 ∣F∣=q|F|=q∣F∣=q。考虑到 xq−xx^{q} - xxq−x 的根在代数闭包中是唯一的，因此 FqF_{q}Fq​ 是唯一的。

结论：

  • 结构链：Fp≅Zp<Fq=FpnF_p \cong \mathbb{Z}_p < F_q = F_{p^n}Fp​≅Zp​<Fq​=Fpn​
  • Fq<FqnF_{q}<F_{q^{n}}Fq​<Fqn​，且 FqnF_{q^{n}}Fqn​ 是 FqF_qFq​ 上的分裂域（从前面推导可以看出，基域的大小不重要）

有限域的 Galois 理论

Galois 群结构

考虑有限扩张 Fqn/FqF_{q^n} / F_qFqn​/Fq​。这是一个 Galois 扩张，其 Galois 群记为 G=Gal(Fqn/Fq)G = Gal(F_{q^n}/F_q)G=Gal(Fqn​/Fq​)。

因此 ∣⟨σ⟩∣=n=∣G∣\left\vert \langle \sigma \rangle \right\vert = n = \left\vert G \right\vert∣⟨σ⟩∣=n=∣G∣，同时 σ∈G⇒⟨σ⟩⊆G\sigma \in G \Rightarrow \langle \sigma \rangle\subseteq Gσ∈G⇒⟨σ⟩⊆G，得到 G=⟨σ⟩G = \langle \sigma \rangleG=⟨σ⟩

对于任意 d>0d > 0d>0，在 FqF_qFq​ 上是否存在次数为 ddd 的不可约多项式？答案是肯定的。

算法与应用：计算多项式的阶

在找到本原多项式之前，先考虑一个问题：给定 FqF_qFq​ 上的 ddd 次不可约多项式 p(x)p(x)p(x)，如何计算其阶（即 p(x)p(x)p(x) 的根的阶 vvv）？

  1. v∣qd−1v \mid q^d - 1v∣qd−1。
  2. αv=1⇒p(x)∣xv−1\alpha ^{v} = 1 \Rightarrow p(x) \mid x^{v} - 1αv=1⇒p(x)∣xv−1

  1. 计算最大可能的周期 N=qd−1N = q^d - 1N=qd−1。
  2. 对 NNN 进行素因子分解：N=p1e1p2e2…pmemN = p_1^{e_1} p_2^{e_2} \dots p_m^{e_m}N=p1e1​​p2e2​​…pmem​​。
  3. 对于每个素因子 pip_ipi​，检查 xN/pi(modp(x))x^{N/p_i} \pmod{p(x)}xN/pi​(modp(x)) 是否为 1：
    • 如果对于某个 pip_ipi​，有 xN/pi≡1(modp(x))x^{N/p_i} \equiv 1 \pmod{p(x)}xN/pi​≡1(modp(x))，说明真正的阶 vvv 是 N/piN/p_iN/pi​ 的因子。
    • 此时令 N←N/piN \leftarrow N/p_iN←N/pi​，重复检查。
    • 如果对于所有的 pip_ipi​，都有 xN/pi≢1(modp(x))x^{N/p_i} \not\equiv 1 \pmod{p(x)}xN/pi​≡1(modp(x))，则当前的 NNN 即为所求的阶 vvv。

实例分析

例：考察 F2F_2F2​ 上的多项式 p(x)=x6+x+1p(x) = x^6 + x + 1p(x)=x6+x+1。
这里 q=2,d=6q=2, d=6q=2,d=6。

  1. 计算最大周期：N=26−1=63N = 2^6 - 1 = 63N=26−1=63。
  2. 分解 NNN：63=32⋅763 = 3^2 \cdot 763=32⋅7。素因子为 3,73, 73,7。
  3. 我们需要检查 N/3=21N/3 = 21N/3=21 和 N/7=9N/7 = 9N/7=9：
    • 计算 x21(modx6+x+1)x^{21} \pmod{x^6+x+1}x21(modx6+x+1)。
    • 计算 x9(modx6+x+1)x^9 \pmod{x^6+x+1}x9(modx6+x+1)。
  4. 若以上结果均不为 1，则 p(x)p(x)p(x) 的阶为 63，即它是一个本原多项式。
