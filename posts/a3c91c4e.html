Introduction and elimination of quantifiers

发表于2025-04-24|更新于2025-07-24|discrete math
|字数总计:1.1k|阅读时长:6分钟

要证明之前用的逻辑推理规则是合理的。

Theorem 1.

ϕ[x↦t]⊨∃xϕ\phi[x\mapsto t]\models\exists x\phi ϕ[x↦t]⊨∃xϕ

where ϕ[x↦t]\phi [ x\mapsto t]ϕ[x↦t] means substituting every free occurrence ofofof xxx in ϕ\phiϕ with ttt.

Example2.
Theorem1 的例子：

P(c)⊨∃x(P(x))P(c)\vDash\exists x(P(x)) P(c)⊨∃x(P(x))

考虑一个更复杂一点的例子：

P(c)∧Q(g(c),c)⊨∃x(P(x)∧Q(g(x),c))P(c)\wedge Q(g(c),c)\models\exists x(P(x)\wedge Q(g(x),c)) P(c)∧Q(g(c),c)⊨∃x(P(x)∧Q(g(x),c))

In order to prove theorem 1, we need to use the following property.
因为 Theorem1 引入了一个新的符号 ϕ[x↦t]\phi[x \mapsto t]ϕ[x↦t]，这个符号表示在语法上把 ϕ\phiϕ 中的 xxx 替换成 ttt。（不包括 ∀x,∃x\forall x, \exists x∀x,∃x 包括到的这些 xxx）

Claim 3.

⟦ϕ[x↦t]⟧J=⟦ϕ⟧J[x↦⟦t⟧J]\llbracket \phi[x \mapsto t] \rrbracket_{\mathcal{J}} = \llbracket \phi \rrbracket_{\mathcal{J}[x \mapsto \llbracket t \rrbracket_{\mathcal{J}}]} [[ϕ[x↦t]]]J​=[[ϕ]]J[x↦[[t]]J​]​

Claim3 可能有点复杂，再举两个例子：

Example 4.

  • Let ϕ=P(x)\phi = P(x)ϕ=P(x) and t=ct=ct=c. Then, ⟦P(c)⟧J=⟦P(x)⟧J[x↦J(c)]\llbracket P(c) \rrbracket_{\mathcal{J}} = \llbracket P(x) \rrbracket_{\mathcal{J}[x \mapsto \mathcal{J}(c)]}[[P(c)]]J​=[[P(x)]]J[x↦J(c)]​
  • Let ϕ=P(x)\phi = P(x)ϕ=P(x) and t=f(y,c)t=f(y,c)t=f(y,c). Then, ⟦P(f(y,c))⟧J=⟦P(x)⟧J[x↦⟦f(y,c)⟧J]\llbracket P(f(y,c)) \rrbracket_{\mathcal{J}} = \llbracket P(x) \rrbracket_{\mathcal{J}[x \mapsto \llbracket f(y,c) \rrbracket_{\mathcal{J}}]}[[P(f(y,c))]]J​=[[P(x)]]J[x↦[[f(y,c)]]J​]​.

上面的 Claim3 直接作为结论来用，不做证明。然后使用 Claim3 来证明 Theorem1：
Proof
For any J\mathcal{J}J, if ⟦ϕ↦t⟧J=T\llbracket \phi \mapsto t \rrbracket_{\mathcal{J}} = T[[ϕ↦t]]J​=T, then ⟦ϕ⟧J[x↦⟦t⟧J]=T\llbracket \phi \rrbracket_{\mathcal{J}[x\mapsto \llbracket t \rrbracket_{\mathcal{J}}]} = T[[ϕ]]J[x↦[[t]]J​]​=T, so ⟦∃xϕ⟧J=T\llbracket \exists x\phi \rrbracket_{\mathcal{J}} = T[[∃xϕ]]J​=T.
Qed.

再考虑一个类似的结论，同样可以用 Claim3 证明。
Theorem 5.

∀xϕ⊨ϕ[x↦t]\forall x \phi \models \phi[x\mapsto t] ∀xϕ⊨ϕ[x↦t]

Proof
If J\mathcal{J}J is an interpretation, ⟦∀xϕ⟧J=T\llbracket \forall x\phi \rrbracket_{\mathcal{J}} = T[[∀xϕ]]J​=T, then for any aaa in J\mathcal{J}J's domain, ⟦ϕ⟧x↦a=T\llbracket \phi \rrbracket_{x\mapsto a} = T[[ϕ]]x↦a​=T.
⟦ϕ[x↦t]⟧J=⟦ϕ⟧J[x↦⟦t⟧J]=T\llbracket \phi[x\mapsto t] \rrbracket_{\mathcal{J}} = \llbracket \phi \rrbracket_{\mathcal{J}\left[x\mapsto \llbracket t \rrbracket_{\mathcal{J}} \right]} = T[[ϕ[x↦t]]]J​=[[ϕ]]J[x↦[[t]]J​]​=T.
Qed.

在之前的逻辑推理中，用过这样的结论：
If Φ,ψ⊢χ\Phi, \psi \vdash \chiΦ,ψ⊢χ and xxx not mentioned in Φ\PhiΦ, then Φ⊢∀x(ψ→χ)\Phi \vdash \forall x(\psi \to \chi)Φ⊢∀x(ψ→χ).
上述结论可以抽象为如下结论

Theorem 7.

  • If Φ⊨ψ\Phi \models \psiΦ⊨ψ and xxx does not freely occur in Φ\PhiΦ, then Φ⊨∀xψ\Phi \models \forall x \psiΦ⊨∀xψ

Remark: “does not freely occur” is very important.

举个例子：
Φ={P(x)},ψ=P(x)\Phi = \{ P(x) \}, \psi = P(x)Φ={P(x)},ψ=P(x)，那么有 Φ⊨ψ\Phi \models \psiΦ⊨ψ，但是 Φ⊭∀xψ\Phi \not \models \forall x \psiΦ⊨∀xψ。

想要证明 Theorem7，需要引入以下 Claim

Claim9
If Φ\PhiΦ does not mention xxx

⟦ϕ⟧J=⟦ϕ⟧J[x↦a]\llbracket \phi \rrbracket_{\mathcal{J}} = \llbracket \phi \rrbracket_{\mathcal{J}[x\mapsto a]} [[ϕ]]J​=[[ϕ]]J[x↦a]​

接下来用 Claim9 证明 Theorem7
Proof
For any J\mathcal{J}J s.t. for any ϕ∈Φ,⟦ϕ⟧J=T\phi \in \Phi, \llbracket \phi \rrbracket_{\mathcal{J}} = Tϕ∈Φ,[[ϕ]]J​=T.
For any aaa in J′s\mathcal{J}'sJ′s domain, and any ϕ∈Φ\phi \in \Phiϕ∈Φ, ⟦ϕ⟧J[x↦a]=⟦ϕ⟧J=T\llbracket \phi \rrbracket_{\mathcal{J}[x\mapsto a]} = \llbracket \phi \rrbracket_{\mathcal{J}} = T[[ϕ]]J[x↦a]​=[[ϕ]]J​=T
So ⟦ψ⟧J[x↦a]=T\llbracket \psi \rrbracket_{\mathcal{J}[x\mapsto a]} = T[[ψ]]J[x↦a]​=T (Φ⊨ψ\Phi \models \psiΦ⊨ψ), ⟦∀xψ⟧J=T\llbracket \forall x \psi \rrbracket_{\mathcal{J}} = T[[∀xψ]]J​=T
Qed.
