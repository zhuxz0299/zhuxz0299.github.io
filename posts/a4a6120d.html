纳什均衡

纳什均衡 (Nash Equilibrium)：用于刻画 nnn 个 agent 的策略
如果 ui(pi,p−i∗)≤ui(pi∗,p−i∗),∀i,piu_i(p_{i},p_{-i}^{*})\le u_i(p_{i}^{*},p_{-i}^{*}), \forall i,p_{i}ui​(pi​,p−i∗​)≤ui​(pi∗​,p−i∗​),∀i,pi​，那么 p∗=(p1∗,p2∗,…,pn∗)p^{*}=(p_1^{*},p_2^{*}, \ldots ,p_n^{*})p∗=(p1∗​,p2∗​,…,pn∗​) 是一个纳什均衡。

由于 pi∗∈BPi(p−i∗)p_{i}^{*}\in BP_{i}(p_{-i}^{*})pi∗​∈BPi​(p−i∗​)，那么可以写出

(p1∗p2∗⋮pn∗)∈(BP1(p−1∗)BP2(p−2∗)⋮BPn(p−n∗))⇒p∗∈BP(p∗)\begin{pmatrix} p_1^{*} \\ p_2^{*} \\ \vdots \\ p_n^{*} \end{pmatrix} \in \begin{pmatrix} BP_1(p_{-1}^{*}) \\ BP_2(p_{-2}^{*}) \\ \vdots \\ BP_{n}(p_{-n}^{*}) \end{pmatrix} \Rightarrow p^{*} \in BP(p^{*}) ​p1∗​p2∗​⋮pn∗​​​∈​BP1​(p−1∗​)BP2​(p−2∗​)⋮BPn​(p−n∗​)​​⇒p∗∈BP(p∗)

因此算子 BPBPBP 如果有不动点(fixed point)，那么存在纳什均衡。在数学上可以证明上述不动点总是存在的，因此纳什均衡也总是有 mixed strategy。

Example

partner game

  • player：p1,p2
  • actions: work hard, be lazy (W,LW,LW,L)
  • utility:
    (p1,p2) WWW LLL
    WWW (10,10) (-5,5)
    LLL (5,-5) (0,0)

满足纳什均衡的 pure strategy 有 (W,W)(W,W)(W,W) 和 (L,L)(L,L)(L,L)。
对于 mixed strategy，设 p1 work 的概率为 xxx, lazy 的概率为 1−x1-x1−x；p2 work 和 lazy 的概率则为 yyy 和 1−y1-y1−y。此时两个 player 的 utility 为：

u1=10xy+(−5)x(1−y)+5(1−x)y+0(1−x)(1−y)=10xy+5y−5x=5x(2y−1)+5yu2=5y(2x−1)+5x\begin{aligned} u_1 &= 10xy+(-5)x(1-y)+5(1-x)y+0(1-x)(1-y) = 10xy+5y-5x = 5x(2y-1) + 5y \\ u_2 &= 5y(2x-1) + 5x \end{aligned} u1​u2​​=10xy+(−5)x(1−y)+5(1−x)y+0(1−x)(1−y)=10xy+5y−5x=5x(2y−1)+5y=5y(2x−1)+5x​

⇒x∗={1  if y > 1/20  if y < 1/2[0,1]  if y = 1/2y∗={1  if x > 1/20  if x < 1/2[0,1]  if x = 1/2\Rightarrow x^{*}=\begin{cases} 1 \text{~ if y > 1/2} \\ 0 \text{~ if y < 1/2} \\ [0,1] \text{~ if y = 1/2} \\ \end{cases} \quad y^{*}=\begin{cases} 1 \text{~ if x > 1/2} \\ 0 \text{~ if x < 1/2} \\ [0,1] \text{~ if x = 1/2} \\ \end{cases} ⇒x∗=⎩⎨⎧​1  if y > 1/20  if y < 1/2[0,1]  if y = 1/2​y∗=⎩⎨⎧​1  if x > 1/20  if x < 1/2[0,1]  if x = 1/2​

由图像可得最终结果为 (1/2,1/2)(1 /2, 1/2)(1/2,1/2)

Dominant & Dominated Strategy

Dominant Strategy

A strategy is a dominant strategy for player iii if

ui(ai,a−i)≤ui(ai∗,a−i)∀ai,a−iu_i(a_i,a_{-i})\le u_i(a_i^{*},a_{-i}) \quad \forall a_i, a_{-i} ui​(ai​,a−i​)≤ui​(ai∗​,a−i​)∀ai​,a−i​

可以看出这个条件比纳什均衡更强。

A strategy profile A=(a1∗,a2∗,…,an∗)A=(a_1^{*},a_2^{*}, \ldots ,a_n^{*})A=(a1∗​,a2∗​,…,an∗​) is a dominant strategy equilibrium if ai∗a_i^{*}ai∗​ is a dominant strategy for each player iii.

DSE(Dominant Strategy Equilibrium)并不是总是存在的，而 NE 却总是存在的。

prisoner dilemma Example

  • player：p1,p2
  • actions: confess, deny (C,DC,DC,D)
  • utility:
    (p1,p2) CCC DDD
    CCC (-3,-3) (0,-5)
    DDD (-5,0) (-1,-1)

在这个例子中，无论对手选择的是什么，选择 CCC 总是对自己更加有利的。因此 (C,C)(C,C)(C,C) 不仅是一个 pure strategy NE，同时也是 Dominant Strategy。

Dominated Strategy

A strategy aia_iai​ is a dominated strategy for player iii if ∃ai′\exists a_i'∃ai′​

ui(ai′,a−i)>ui(ai,a−i)∀a−iu_i(a_i',a_{-i}) > u_i(a_i,a_{-i}) \quad \forall a_{-i} ui​(ai′​,a−i​)>ui​(ai​,a−i​)∀a−i​

因此假如一个 game 里存在 dominated strategy，我们就可以直接将这个策略去掉。

prisoner dilemma Example

还是上面的例子，但是让第一个囚徒加上自杀(suicide)的选项：

(p1,p2) CCC DDD
CCC (-3,-3) (0,-5)
DDD (-5,0) (-1,-1)
S (-100,-3) (-100,0)

此时 SSS 对于 CCC 来说就是一个 dominated strategy。

Correlated Equilibrium

Let P(a)P(a)P(a) be a (joint) probability distribution over a∈A1×A2×⋯×Ana\in A_1\times A_2\times \cdots \times A_na∈A1​×A2​×⋯×An​, then P∗(a)P^{*}(a)P∗(a) is a correlated equilibrium (CE) if

∑a−ip∗(a−i∣ai)×ui(ai,a−i)≥∑a−ip∗(a−i∣ai)×ui(ai′,a−i)∀ai,ai′\sum_{a_{-i}}p^{*}(a_{-i}|a_i)\times u_i(a_i,a_{-i}) \ge \sum_{a_{-i}} p^{*}(a_{-i}|a_i)\times u_i(a_i',a_{-i}) \quad \forall a_i,a_i' a−i​∑​p∗(a−i​∣ai​)×ui​(ai​,a−i​)≥a−i​∑​p∗(a−i​∣ai​)×ui​(ai′​,a−i​)∀ai​,ai′​

这里表示所有的 player 都没有动机去 deviate。

上面的式子经过数学变换之后同样可以写为

∑a−ip∗(a−i,ai)×ui(ai,a−i)≥∑a−ip∗(a−i,ai)×ui(ai′,a−i)∀ai,ai′\sum_{a_{-i}}p^{*}(a_{-i}, a_i)\times u_i(a_i,a_{-i}) \ge \sum_{a_{-i}} p^{*}(a_{-i}, a_i)\times u_i(a_i',a_{-i}) \quad \forall a_i,a_i' a−i​∑​p∗(a−i​,ai​)×ui​(ai​,a−i​)≥a−i​∑​p∗(a−i​,ai​)×ui​(ai′​,a−i​)∀ai​,ai′​

我们希望能够调整 P(a)P(a)P(a) 使得结果更优，因此这是个优化问题：

max⁡∑p∗(a)ui(a)∑a∈Ap∗(a)=1p∗(a)≥0\max_{} \sum_{} p^{*}(a) u_i(a) \\ \sum_{a\in A} p^{*}(a)=1 \\ p^{*}(a)\ge 0 max​∑​p∗(a)ui​(a)a∈A∑​p∗(a)=1p∗(a)≥0

可以看出这是个线性规划，有多项式复杂度的方法可以求解。但是 MNE(Mixed Nash Equilibrium) 是没有特定解法的。因此 Correlated Equilibrium 更弱。

Traffic light game Example

Example1

(p1,p2) GoGoGo YieldYieldYield
GoGoGo (-10,-10) (5,0)
YieldYieldYield (0,5) (-1,-1)

可以得到

  • pure NE: (G,Y)(G,Y)(G,Y), (Y,G)(Y,G)(Y,G)
  • mixed NE: x=Pr⁡(player 1 plays G),y=Pr⁡(player 2 plays G)x=\operatorname{Pr}(\text{player 1 plays G}), y=\operatorname{Pr}(\text{player 2 plays G})x=Pr(player 1 plays G),y=Pr(player 2 plays G)
    • x∗=y∗=3/8x^{*}=y^{*}=3 /8x∗=y∗=3/8
  • the expected payoff to p1 is −15/32-15 /32−15/32

上述情况是两个人的选择是独立的。但是假如出现了红绿灯，那么就会影响两个 player 的选择。假设

Pr⁡(G,Y)=0.5Pr⁡(Y,G)=0.5\operatorname{Pr}(G,Y)=0.5 \quad \operatorname{Pr}(Y,G)=0.5 Pr(G,Y)=0.5Pr(Y,G)=0.5

其中 (G,Y)(G,Y)(G,Y) 表示红绿灯推荐 player 1 往前走，player 2 停下。

那么此时我们需要考虑两个 player 是否会遵守红绿灯。

  • If p1 is told to play GGG, it knows that p2 is playing YYY
  • If p1 is told to play YYY, it knows that p2 is playing GGG

the expected payoff to p1 is 3/23/23/2

可以得知两个 player 都会遵守。

Example2

假如红绿灯的情况更加复杂

Pr⁡(G,Y)=0.55Pr⁡(Y,G)=0.4Pr⁡(Y,Y)=0.05\operatorname{Pr}(G,Y)=0.55 \quad \operatorname{Pr}(Y,G)=0.4 \quad \operatorname{Pr}(Y,Y)=0.05 Pr(G,Y)=0.55Pr(Y,G)=0.4Pr(Y,Y)=0.05

  • If p1 is told to play GGG, it knows that p2 is playing YYY
  • If p1 is told to play YYY, it knows the probability of p2 playing YYY is 1/91/91/9; GGG is 8/98 /98/9。
    • If p1 stick to YYY, its expected payoff is 1/9×(−1)+8/9×0=−1/91 /9 \times (-1) + 8 /9 \times 0 = -1/ 91/9×(−1)+8/9×0=−1/9
    • If p1 stick to YYY, its expected payoff is 1/9×5+8/9×(−10)=−75/91 /9 \times 5 + 8 /9 \times (-10) = -75/ 91/9×5+8/9×(−10)=−75/9

因此 player 仍然会选择 YYY。
