FOL semantics

可以由一阶逻辑结构(SSS-structures)来决定真假的命题：
∀x(L(x,f(x)))∀x(M(x))∃x(P(x)∧E(x))G(c)\forall x(L(x, f(x)))\quad \forall x(M(x))\quad \exists x(P(x)\land E(x)) \quad G(c)∀x(L(x,f(x)))∀x(M(x))∃x(P(x)∧E(x))G(c)

这些命题的真假和变量符号 xxx 的具体取值无关，而是与 ∀\forall∀ 或者 ∃\exists∃ 描述的范围范围有关。但是有些一阶逻辑命题也是可以没有量词的，比如：
L(x,y)L(x,f(x))L(x, y) \quad L(x,f(x))L(x,y)L(x,f(x))

但是这些命题的真与假就取决于变量的赋值。

Definition 1 (Free occurrence and binded occurrence，约束出现和自由出现). Assume xxx is a variable symbol. In ∀xϕ\forall x \phi∀xϕ and ∃xϕ\exists x \phi∃xϕ, all occurrences of xxx are called binded occurrences. All other occurrences of xxx are free occurrences.

例如上述 L(x,y)L(x, y)L(x,y) 中的 x,yx, yx,y 都是自由出现；但如果是 ∀x∃y(L(x,y))\forall x\exists y(L(x, y))∀x∃y(L(x,y))，那么 x,yx, yx,y 就都是约束出现。

Definition 2 (Open proposition and closed proposition，开命题和闭命题). A FOL proposition ϕ\phiϕ is open if it contains at least one free occurrence of variables. Otherwise, ϕ\phiϕ is called a closed proposition.

也就是说只要命题中至少有一个自由出现的变量符号，那这个一阶逻辑命题就是开命题。根据上节课的内容，对于闭命题而言，只要通过一阶逻辑的结构（SSS-structure）就可以判断命题的真值。但如果是开命题，就还和变量值有关。

Definition 4 (SSS-interpretation，SSS 解释).
Given a symbol set SSS, a SSS-interpretation J=(A,β)\mathcal{J} = (\mathcal{A}, \beta)J=(A,β) is

  • a SSS-structure A=(A,α)\mathcal{A} = (A, \alpha)A=(A,α)
  • a SSS-assignment β\betaβ: a mapping from variables to elements in the domain AAA
    For J=(A,β)\mathcal{J} = (\mathcal{A}, \beta)J=(A,β) and A=(A,α)\mathcal{A} = (A, \alpha)A=(A,α), we usually use J(P)\mathcal{J}(P)J(P) and A(P)\mathcal{A}(P)A(P) to represent α(P)\alpha(P)α(P), use J(f)\mathcal{J}(f)J(f) and A(f)\mathcal{A}(f)A(f) to represent α(f)\alpha(f)α(f), use J(c)\mathcal{J}(c)J(c) and A(c)\mathcal{A}(c)A(c) to represent α(c)\alpha(c)α(c) and use J(x)\mathcal{J}(x)J(x) to represent β(x)\beta(x)β(x).

就是说 SSS-interpretation 就是在 SSS-structure 的基础上加了一个对变量符号的解释，即把每一个变量符号都映射到论域中的元素。通常用字母 J\mathcal{J}J 来表示一个解释。

Defination 5 (Terms’ denotation). For SSS-interpretation J\mathcal{J}J and a SSS-term ttt,

  • [ ⁣[x] ⁣]J=J(x)[\![x]\!]_{\mathcal{J}} = \mathcal{J}(x)[[x]]J​=J(x)
  • [ ⁣[c] ⁣]J=J(c)[\![c]\!]_{\mathcal{J}} = \mathcal{J}(c)[[c]]J​=J(c)
  • [ ⁣[f(t1,t2,…,tn)] ⁣]J=J(f)([ ⁣[t1] ⁣]J,[ ⁣[t2] ⁣]J,…[ ⁣[tn] ⁣]J)[\![f(t_1, t_2, \ldots ,t_n)]\!]_{\mathcal{J}} = \mathcal{J}(f) ([\![t_1]\!]_{\mathcal{J}}, [\![t_2]\!]_{\mathcal{J}}, \ldots [\![t_n]\!]_{\mathcal{J}})[[f(t1​,t2​,…,tn​)]]J​=J(f)([[t1​]]J​,[[t2​]]J​,…[[tn​]]J​)

这里定义了 term 的语义，方便后面定义命题真值。

Definition 6 (Propositions’ truth).
For SSS-interpretation J\mathcal{J}J and a SSS-proposition ttt,

  • [ ⁣[P(t1,t2,…,tn)] ⁣]J=J(P)([ ⁣[t1] ⁣]J,[ ⁣[t2] ⁣]J,…,[ ⁣[tn] ⁣]J)[\![\mathrm{P}\left(t_{1}, t_{2}, \ldots, t_{n}\right)]\!]_{\mathcal{J}}=\mathcal{J}(P)\left([\![t_{1}]\!]_{\mathcal{J}},[\![t_{2}]\!]_{\mathcal{J}}, \ldots,[\![t_{n}]\!]_{\mathcal{J}}\right)[[P(t1​,t2​,…,tn​)]]J​=J(P)([[t1​]]J​,[[t2​]]J​,…,[[tn​]]J​)
  • [ ⁣[ϕ∧ψ] ⁣]J=[ ⁣[∧] ⁣]([ ⁣[ϕ] ⁣]J,[ ⁣[ψ] ⁣]J)[\![\phi \wedge \psi]\!]_{\mathcal{J}}=[\![\wedge]\!]\left([\![\phi]\!]_{\mathcal{J}},[\![\psi]\!]_{\mathcal{J}}\right)[[ϕ∧ψ]]J​=[[∧]]([[ϕ]]J​,[[ψ]]J​)
  • [ ⁣[¬ϕ] ⁣]J=[ ⁣[¬] ⁣]([ ⁣[ϕ] ⁣]J)[\![\neg \phi]\!]_{\mathcal{J}}=[\![\neg]\!]\left([\![\phi]\!]_{\mathcal{J}}\right)[[¬ϕ]]J​=[[¬]]([[ϕ]]J​)
  • [ ⁣[∀xϕ] ⁣]J=T[\![\forall x \phi]\!]_{\mathcal{J}}=\mathbf{T}[[∀xϕ]]J​=T if and only if for every aaa in A\mathcal{A}A 's domain, [ ⁣[ϕ] ⁣]J[x↦a]=T[\![\phi]\!]_{\mathcal{J}[x \mapsto a]}=\mathbf{T}[[ϕ]]J[x↦a]​=T
  • [ ⁣[∃xϕ] ⁣]J=T[\![\exists x \phi]\!]_{\mathcal{J}}=\mathbf{T}[[∃xϕ]]J​=T if and only if for at least one aaa in A\mathcal{A}A 's domain, [ ⁣[ϕ] ⁣]J[x↦a]=T[\![\phi]\!]_{\mathcal{J}[x \mapsto a]}=\mathbf{T}[[ϕ]]J[x↦a]​=T

where J[x↦a]\mathcal{J}[x \mapsto a]J[x↦a] is a S-interpretation which keeps all other interpretations in J\mathcal{J}J and interprets xxx by aaa.

Properties of first order propositions

Definition 9. Suppose Φ\PhiΦ is a set of SSS-propositions and ψ\psiψ is one single SSS-proposition. We say that ψ\psiψ is a consequence (语义后承) of Φ\PhiΦ, written as Φ⊨ψ\Phi \models \psiΦ⊨ψ, if for any SSS-interpretation J\mathcal{J}J, [ ⁣[ϕ] ⁣]J=T[\![\phi]\!]_{\mathcal{J}} = \mathbf{T}[[ϕ]]J​=T for any ϕ∈Φ\phi \in \Phiϕ∈Φ implies [ ⁣[ψ] ⁣]J=T[\![\psi]\!]_{\mathcal{J}} = \mathbf{T}[[ψ]]J​=T.

Example 10. G(c)⊨∃x(G(x))G(c) \models \exists x(G(x))G(c)⊨∃x(G(x)).
Proof.
Assume J\mathcal{J}J is an interpretation, s.t. [ ⁣[G(c)] ⁣]J=T[\![G(c)]\!]_{\mathcal{J}} = \mathbf{T}[[G(c)]]J​=T. In other words, J(G)(J(c))=T\mathcal{J}(G)(\mathcal{J}(c)) = \mathbf{T}J(G)(J(c))=T.
On the other hand [ ⁣[∃x(G(x))] ⁣]J=T[\![\exists x(G(x))]\!]_{\mathcal{J}} = \mathbf{T}[[∃x(G(x))]]J​=T iff. there exists at least an element aaa in J\mathcal{J}J's domain, s.t. [ ⁣[G(x)] ⁣]J[x↦a]=T[\![G(x)]\!]_{\mathcal{J}[x \mapsto a]} = \mathbf{T}[[G(x)]]J[x↦a]​=T, i.e. J(G)(a)=T\mathcal{J}(G)(a) = \mathbf{T}J(G)(a)=T.
This is obviously because we can pick a=J(c)a = \mathcal{J}(c)a=J(c).
Qed.

Example 11. ∀x(M(x))⊨M(c)\forall x(M(x)) \models M(c)∀x(M(x))⊨M(c).
Proof.
Assume J\mathcal{J}J is an interpretation, s.t. [ ⁣[∀x(M(x))] ⁣]J=T[\![\forall x(M(x))]\!]_{\mathcal{J}} = \mathbf{T}[[∀x(M(x))]]J​=T. That means, for every aaa in J\mathcal{J}J's domain,
[ ⁣[M(x)] ⁣]J[x↦a]=T[\![M(x)]\!]_{\mathcal{J}[x \mapsto a]} = \mathbf{T}[[M(x)]]J[x↦a]​=T,
i.e. J(M)(a)=T\mathcal{J}(M)(a) = \mathbf{T}J(M)(a)=T.
Therefore, [ ⁣[M(c)] ⁣]J=J(M)(J(c))=T[\![M(c)]\!]_{\mathcal{J}} = \mathcal{J}(M)(\mathcal{J}(c)) = \mathbf{T}[[M(c)]]J​=J(M)(J(c))=T.
Qed.
