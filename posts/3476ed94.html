常用stl容器的常用操作及其时间复杂度

map

基于红黑树实现

  • 插入键值对，时间为 O(log⁡n)O(\log n)O(logn)。使用 operator[] 或 insert 函数可以插入键-值对。如果键已经存在，operator[] 会更新对应的值，而 insert 则不会更新。
  • 访问和修改值，时间为 O(log⁡n)O(\log n)O(logn)。使用 operator[] 可以通过键访问和修改对应的值。
  • 检查键是否存在，时间为 O(log⁡n)O(\log n)O(logn)。
    • 使用 count 函数可以检查给定键在 std::map 中的出现次数。如果键存在，返回值大于 000；不存在则返回 000。
    • 使用 find 函数可以检查给定键是否存在。如果键存在，返回指向该键-值对的迭代器；不存在则返回 std::map::end()。
  • 删除键-值对，时间为 O(log⁡n)O(\log n)O(logn)。使用 erase 函数可以删除给定键的键-值对。
  • 获取键-值对个数，时间为 O(1)O(1)O(1)。使用 size 函数可以获取 std::map 中键-值对的个数。
  • 清空 std::map，时间为 O(n)O(n)O(n)。使用 clear 函数可以清空 std::map，将其变为空。
  • 遍历 std::map，时间为 O(n)O(n)O(n)。使用迭代器可以遍历 std::map 中的所有键-值对。

set

基于红黑树实现

  • 插入元素（insert）：平均时间复杂度为 O(logn)O(log n)O(logn)
  • 删除元素（erase）：平均时间复杂度为 O(logn)O(log n)O(logn)
  • 查找元素（find）：平均时间复杂度为 O(logn)O(log n)O(logn)
  • 获取元素个数（size）：O(1)O(1)O(1)。
  • 判断容器是否为空（empty）：O(1)O(1)O(1)。
  • 遍历所有元素（使用迭代器）：O(n)O(n)O(n)。

vector

  • 插入元素（push_back()）
    • 平均时间复杂度: O(1)O(1)O(1)
    • 最坏时间复杂度: O(n)O(n)O(n)，当需要重新分配内存时
  • 访问元素（使用索引）,平均时间复杂度: O(1)O(1)O(1)
  • 删除元素（pop_back() 或 erase()），时间复杂度分别为 O(1)O(1)O(1) 和 O(n)O(n)O(n)
  • 获取元素个数（size()）：O(1)O(1)O(1)。
  • 判断容器是否为空（empty()）：O(1)O(1)O(1)。
  • 遍历所有元素（使用迭代器）：O(n)O(n)O(n)。
  • 在指定位置插入元素（insert()），平均时间复杂度: O(n)O(n)O(n)
  • 清空容器（clear()）：时间复杂度: O(1)O(1)O(1)

vector 的创建与初始化

  • 创建一维 vector
    • 使用默认构造函数创建空的 std::vector，然后使用 push_back() 方法逐个添加元素。
    • 使用初始化列表初始化 std::vector。
    • 使用构造函数并指定元素数量和默认值。
  • 创建二维 vector
    • 使用默认构造函数和嵌套的 std::vector 创建一个空的二维 std::vector，然后使用嵌套的 push_back() 方法逐行添加元素。
    • 使用初始化列表和嵌套的初始化列表初始化二维 std::vector。
    • 使用构造函数和嵌套的 std::vector，并指定行数和列数。

array

  • 访问元素：使用下标操作符 [] 或 at() 方法可以访问指定位置的元素。时间复杂度：O(1)O(1)O(1)
  • 获取元素个数（size()）：O(1)O(1)O(1)。
  • 遍历所有元素（使用迭代器）：O(n)O(n)O(n)。
  • 填充数组：fill() 方法可以将数组的所有元素设置为指定的值。时间复杂度：O(n)O(n)O(n)
  • 比较操作：== 和 != 操作符可用于比较两个数组是否相等。
