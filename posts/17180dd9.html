Propositional Logic and Logic Connectives

  • 首先是引入了三个逻辑连接词 ∧,∨,¬\land , \lor , \lnot∧,∨,¬。
    • 优先级：¬,∧,∨\lnot , \land , \lor¬,∧,∨
  • 然后还有原子命题：p,q,rp, q, rp,q,r。
  • 利用逻辑连接词、原子命题还有适当的括号，就能构成复合命题。

Defination 2. Suppose Σ\SigmaΣ is the set of propositional variables. A mapping J:Σ→{T,F}\mathcal{J}:\Sigma \to \{ T, F \}J:Σ→{T,F} called a truth assignment (真值指派).

例如 Σ={p,q,r}\Sigma = \{ p, q, r \}Σ={p,q,r}, J(p)=T,J(q)=F,J(r)=F\mathcal{J}(p) = T, \mathcal{J}(q) = F, \mathcal{J}(r) = FJ(p)=T,J(q)=F,J(r)=F。在没有真值指派的情况下，原子命题的真与假没有意义。

Definition 3. Suppose Σ\SigmaΣ is the set of propositional variables and J:Σ→{T,F}\mathcal{J}:\Sigma \to \{ T, F \}J:Σ→{T,F} is a truth assignment. The truth value of compound proposition ϕ\phiϕ on J\mathcal{J}J (written as [ ⁣[ϕ] ⁣]J[\![\phi]\!]_{\mathcal{J}}[[ϕ]]J​ ), is defined recursively as follows:

  • [ ⁣[p] ⁣]J=J(p)[\![p]\!]_{\mathcal{J}} = \mathcal{J}(p)[[p]]J​=J(p) for atomic propositions. 就是用真值指派本身定义。
  • [ ⁣[ϕ∧ψ] ⁣]J=[ ⁣[∧] ⁣]([ ⁣[ϕ] ⁣]J,[ ⁣[ψ] ⁣]J)[\![\phi \land \psi]\!]_{\mathcal{J}} = [\![\land ]\!] ([\![\phi]\!]_{\mathcal{J}}, [\![\psi]\!]_{\mathcal{J}})[[ϕ∧ψ]]J​=[[∧]]([[ϕ]]J​,[[ψ]]J​)
  • [ ⁣[ϕ∨ψ] ⁣]J=[ ⁣[∨] ⁣]([ ⁣[ϕ] ⁣]J,[ ⁣[ψ] ⁣]J)[\![\phi \lor \psi]\!]_{\mathcal{J}} = [\![\lor ]\!] ([\![\phi]\!]_{\mathcal{J}}, [\![\psi]\!]_{\mathcal{J}})[[ϕ∨ψ]]J​=[[∨]]([[ϕ]]J​,[[ψ]]J​)
  • [ ⁣[¬ϕ] ⁣]J=[ ⁣[¬] ⁣]([ ⁣[ϕ] ⁣]J)[\![\lnot \phi]\!]_{\mathcal{J}} = [\![\lnot ]\!] ([\![\phi]\!]_{\mathcal{J}})[[¬ϕ]]J​=[[¬]]([[ϕ]]J​)

where [ ⁣[∧] ⁣],[ ⁣[∨] ⁣][\![\land ]\!], [\![\lor ]\!][[∧]],[[∨]] and [ ⁣[¬] ⁣][\![\lnot ]\!][[¬]] represents truth tables of ∧,∨\land , \lor∧,∨ and ¬\lnot¬.

  • 以上定义了复合命题在真值指派上的真值。这里强调了复合命题就是符合了某些语法（原子命题和逻辑连接词的特定组合）的一串符号，不要把它看成真值。
  • 想要知道复合命题在任意真值指派下的真值，使用真值表是可行的办法。

Logical Equivalence

Definition 4. ϕ\phiϕ is a logically equivalent (逻辑等价) to ψ\psiψ, written as ϕ≡ψ\phi \equiv \psiϕ≡ψ, if ϕ\phiϕ’s truth value and ψ\psiψ’s truth value are the same under any situation. In other words, ϕ≡ψ\phi\equiv \psiϕ≡ψ if [ ⁣[ϕ] ⁣]J=[ ⁣[ψ] ⁣]J[\![\phi]\!]_{\mathcal{J}} = [\![\psi]\!]_{\mathcal{J}}[[ϕ]]J​=[[ψ]]J​ for any truth assignment J\mathcal{J}J.

  • 注意，逻辑等价并不说明两个命题是同一个命题。两个命题只是从真值的角度区分不了，但是从语法形式的角度可以区分。
  • 证明逻辑等价可以使用真值表说明，如果在所有真值指派下两个命题真值相等，那么逻辑等价。

Theorem 6

  • ¬(¬ϕ)≡ϕ\lnot ( \lnot \phi ) \equiv \phi¬(¬ϕ)≡ϕ (Double Negation)
  • ϕ∨ϕ≡ϕ\phi \lor \phi \equiv \phiϕ∨ϕ≡ϕ, ϕ∧ϕ≡ϕ\phi \land \phi \equiv \phiϕ∧ϕ≡ϕ (Idempotent Laws, 幂等律)
  • ϕ∨ψ≡ψ∨ϕ\phi \lor \psi \equiv \psi \lor \phiϕ∨ψ≡ψ∨ϕ, ϕ∧ψ≡ψ∧ϕ\phi \land \psi \equiv \psi \land \phiϕ∧ψ≡ψ∧ϕ (Commutative Laws, 交换律)
  • (ϕ∨ψ)∨χ≡ϕ∨(ψ∨χ)( \phi \lor \psi ) \lor \chi \equiv \phi \lor ( \psi \lor \chi )(ϕ∨ψ)∨χ≡ϕ∨(ψ∨χ),
    (ϕ∧ψ)∧χ≡ϕ∧(ψ∧χ)( \phi \land \psi ) \land \chi \equiv \phi \land ( \psi \land \chi )(ϕ∧ψ)∧χ≡ϕ∧(ψ∧χ) (Associative Laws, 结合律)
  • ϕ∨(ψ∧χ)≡(ϕ∨ψ)∧(ϕ∨χ)\phi\lor(\psi\land\chi)\equiv(\phi\lor\psi)\land(\phi\lor\chi)ϕ∨(ψ∧χ)≡(ϕ∨ψ)∧(ϕ∨χ)
    ϕ∧(ψ∨χ)≡(ϕ∧ψ)∨(ϕ∧χ)\phi \land ( \psi \lor \chi ) \equiv ( \phi \land \psi ) \lor ( \phi \land \chi )ϕ∧(ψ∨χ)≡(ϕ∧ψ)∨(ϕ∧χ) ( Distributive Laws,分配律)
  • ¬(ϕ∧ψ)≡(¬ϕ)∨(¬ψ)\lnot ( \phi \land \psi ) \equiv ( \lnot \phi ) \lor ( \lnot \psi )¬(ϕ∧ψ)≡(¬ϕ)∨(¬ψ),
    ¬(ϕ∨ψ)≡(¬ϕ)∧(¬ψ)\lnot ( \phi \lor \psi ) \equiv ( \lnot \phi ) \land ( \lnot \psi )¬(ϕ∨ψ)≡(¬ϕ)∧(¬ψ) (De Morgan’s Laws)

Theorem 7 (Negation Laws)
有一些命题在任意的真值指派上都为真或者都为假，那么在逻辑等价的推理的时候会用 T,FT, FT,F 表示恒为真或者恒为假的命题。注意，这里并不是表示命题和真值逻辑等价。

  • ϕ∨(¬ϕ)≡T\phi \lor ( \lnot \phi ) \equiv Tϕ∨(¬ϕ)≡T
  • ϕ∧(¬ϕ)≡F\phi \land (\lnot\phi)\equiv Fϕ∧(¬ϕ)≡F

Theorem 8 (Laws of logical constants).

  • ϕ∧T≡ϕ\phi \land T \equiv \phiϕ∧T≡ϕ
  • ϕ∨F≡ϕ\phi \lor F \equiv \phiϕ∨F≡ϕ
  • ϕ∨T≡T\phi \lor T \equiv Tϕ∨T≡T
  • ϕ∧F≡F\phi \land F \equiv Fϕ∧F≡F

Theorem 9 (Absorption Laws).

  • ϕ∨(ϕ∧ψ)≡ϕ\quad\phi\lor(\phi\land\psi)\equiv\phiϕ∨(ϕ∧ψ)≡ϕ
  • ϕ∧(ϕ∨ψ)≡ϕ\quad\phi\land(\phi\lor\psi)\equiv\phiϕ∧(ϕ∨ψ)≡ϕ

Theorem 10 (Transitivity(传递性))
If ϕ≡ψ\phi\equiv\psiϕ≡ψ and ψ≡χ\psi\equiv\chiψ≡χ,then ϕ≡χ\phi\equiv\chiϕ≡χ.

Theorem 11 (Congruence Property(算子对于逻辑等价的保持)).

  • If ϕ≡ψ\phi\equiv\psiϕ≡ψ then ¬ϕ≡¬ψ\lnot\phi\equiv\lnot\psi¬ϕ≡¬ψ.
  • If ϕ1≡ϕ2\phi_{1}\equiv\phi_{2}ϕ1​≡ϕ2​ and ψ1≡ψ2\psi_{1}\equiv\psi_{2}ψ1​≡ψ2​ then ϕ1∧ψ1≡ϕ2∧ψ2\phi_{1}\land\psi_{1}\equiv\phi_{2}\land\psi_{2}ϕ1​∧ψ1​≡ϕ2​∧ψ2​.
  • If ϕ1≡ϕ2\phi_{1}\equiv\phi_{2}ϕ1​≡ϕ2​ and ψ1≡ψ2\psi_{1}\equiv\psi_{2}ψ1​≡ψ2​ then ϕ1∨ψ1≡ϕ2∨ψ2\phi_{1}\lor\psi_{1}\equiv\phi_{2}\lor\psi_{2}ϕ1​∨ψ1​≡ϕ2​∨ψ2​.

Theorem 12 (Reflexivity(自反性)). ϕ≡ϕ\phi\equiv\phiϕ≡ϕ.

Consequence Relation

Definition 13. Suppose Φ\PhiΦ is a set of propositions and ψ\psiψ is one single proposition. We say that ψ\psiψ is a consequence (语义后承) of Ψ\PsiΨ, written as Ψ⊨ϕ\Psi \models \phiΨ⊨ϕ, if Ψ\PsiΨ’s being all true implies that ψ\psiψ is also true. In other words, Ψ⊨ψ\Psi \models \psiΨ⊨ψ if for any truth assignment J\mathcal{J}J, [ ⁣[ϕ] ⁣]J=T[\![\phi]\!]_{\mathcal{J}}= T[[ϕ]]J​=T for any ϕ∈Ψ\phi \in \Psiϕ∈Ψ implies [ ⁣[ψ] ⁣]J=T[\![\psi]\!]_{\mathcal{J}}=T[[ψ]]J​=T.
We will write ⊨ψ,ϕ⊨ψ,ϕ1,ϕ2,…,ϕn\models \psi, \phi \models \psi, \phi_1, \phi_2, \ldots ,\phi_{n}⊨ψ,ϕ⊨ψ,ϕ1​,ϕ2​,…,ϕn​ and Φ,ϕ1,ϕ2,…ϕn⊨ψ\Phi, \phi_1, \phi_2, \ldots \phi_{n} \models \psiΦ,ϕ1​,ϕ2​,…ϕn​⊨ψ to represent ∅⊨ψ,{ϕ}⊨ψ,{ϕ1,ϕ2,…,ϕn}⊨ψ\varnothing \models \psi, \{ \phi \}\models \psi , \{ \phi_1, \phi_2, \ldots ,\phi_{n} \}\models \psi∅⊨ψ,{ϕ}⊨ψ,{ϕ1​,ϕ2​,…,ϕn​}⊨ψ and Φ∪{ϕ1,ϕ2,…,ϕn}⊨ψ\Phi \cup \{ \phi_1, \phi_2, \ldots ,\phi_{n} \} \models \psiΦ∪{ϕ1​,ϕ2​,…,ϕn​}⊨ψ.
