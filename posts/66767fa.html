分圆扩张的定义与背景

我们讨论在域 FFF 上的多项式 xn−1x^n - 1xn−1 的分裂域 SSS。为了保证根的性质良好（即没有重根），我们首先需要对域的特征及其与 nnn 的关系做出限定。

前提条件
假定域 FFF 的特征 char(F)∤n\text{char}(F) \nmid nchar(F)∤n。

  • 如果 char(F)=p>0\text{char}(F) = p > 0char(F)=p>0，则要求 p∤np \nmid np∤n。
  • 如果 char(F)=0\text{char}(F) = 0char(F)=0，则无限制。

在这种条件下，(xn−1)′=nxn−1≠0(x^n - 1)' = n x^{n-1} \neq 0(xn−1)′=nxn−1=0，且与 xn−1x^n - 1xn−1 互素（因为两者没有公共根），因此 xn−1x^n - 1xn−1 在分裂域中有 nnn 个不同的根。

典型例子：

  • F=QF = \mathbb{Q}F=Q，则 S⊆CS \subseteq \mathbb{C}S⊆C，根的形式为 e2kπine^{\frac{2k\pi i}{n}}en2kπi​。
  • FFF 为有限域时，即为有限域的扩张。

单位根群的结构

记 UnU_nUn​ 为 xn−1x^n - 1xn−1 在分裂域 SSS 中的所有根的集合：

Un={α∈S∣αn=1}U_n = \{ \alpha \in S \mid \alpha^n = 1 \} Un​={α∈S∣αn=1}

我们已知 ∣Un∣=n|U_n| = n∣Un​∣=n。

为了确定 UnU_nUn​ 的群结构，我们需要引入一个重要的代数引理。

引理：有限子群定理
设 FFF 是一个域。若 GGG 是 F∗F^*F∗（即 F∖{0}F \setminus \{0\}F∖{0}，域的乘法群）的有限子群，则 GGG 是循环群。

注：有限域 FFF 的乘法群 F∗F^*F∗ 本身是循环群，即为该定理的一个特例。

定理：单位根群结构
UnU_nUn​ 关于乘法构成一个循环群。
它的结构同构于模 nnn 的整数加法群：

(Un,⋅)≅(Zn,+)(U_n, \cdot) \cong (\mathbb{Z}_n, +) (Un​,⋅)≅(Zn​,+)

定理证明过程：
我们要证明 (Un,⋅)(U_n, \cdot)(Un​,⋅) 是循环群。由上述引理，我们只需证明 UnU_nUn​ 是 S∗S^*S∗ 的子群即可。

  1. 子群判定：对于任意 α,β∈Un\alpha, \beta \in U_nα,β∈Un​，我们需要验证 αβ−1∈Un\alpha\beta^{-1} \in U_nαβ−1∈Un​。

    (αβ−1)n=αn(βn)−1=1⋅1−1=1(\alpha\beta^{-1})^n = \alpha^n (\beta^n)^{-1} = 1 \cdot 1^{-1} = 1 (αβ−1)n=αn(βn)−1=1⋅1−1=1

    因为 (αβ−1)n=1(\alpha\beta^{-1})^n = 1(αβ−1)n=1，所以 αβ−1\alpha\beta^{-1}αβ−1 也是 xn−1x^n - 1xn−1 的根，即 αβ−1∈Un\alpha\beta^{-1} \in U_nαβ−1∈Un​。

  2. 结论：UnU_nUn​ 是 S∗S^*S∗ 的有限子群，根据引理，UnU_nUn​ 必定是循环群。

本原单位根 (Primitive Roots)

由于 UnU_nUn​ 是循环群，我们可以选取它的生成元。

定义：本原单位根
UnU_nUn​ 的生成元称为 111 的本原 nnn 次根。记 ωn\omega_nωn​ 为 111 的某一个本原 nnn 次根，则有：

Un=⟨ωn⟩={1,ωn,ωn2,…,ωnn−1}U_n = \langle \omega_n \rangle = \{ 1, \omega_n, \omega_n^2, \dots, \omega_n^{n-1} \} Un​=⟨ωn​⟩={1,ωn​,ωn2​,…,ωnn−1​}

元素的阶 (Order)：
对于任意 ωnk∈Un\omega_n^k \in U_nωnk​∈Un​：

ord(ωnk)=n  ⟺  gcd⁡(n,k)=1\text{ord}(\omega_n^k) = n \iff \gcd(n, k) = 1 ord(ωnk​)=n⟺gcd(n,k)=1

记 Ωn\Omega_nΩn​ 为所有本原 nnn 次根的集合：

Ωn={ωnk∣1≤k≤n,gcd⁡(n,k)=1}\Omega_n = \{ \omega_n^k \mid 1 \le k \le n, \gcd(n, k) = 1 \} Ωn​={ωnk​∣1≤k≤n,gcd(n,k)=1}

其元素个数为欧拉函数 ∣Ωn∣=ϕ(n)|\Omega_n| = \phi(n)∣Ωn​∣=ϕ(n)。

同时可以发现 Ωn\Omega_nΩn​ 与 Zn∗\mathbb{Z}_n^{*}Zn∗​ 的元素存在一一对应关系：ωnk↔k\omega_n^{k} \leftrightarrow kωnk​↔k

分圆扩张与伽罗瓦群

定义：分圆扩张 (Cyclotomic Extension)
域 FFF 上 xn−1x^n - 1xn−1 的分裂域 SSS 称为 FFF 上的 nnn 阶分圆扩张。

我们可以将其写为单扩张形式：

S=F(Un)=F(ωn)S = F(U_n) = F(\omega_n) S=F(Un​)=F(ωn​)

其中 ωn\omega_nωn​ 是任意一个本原 nnn 次根。

注：关于生成元的选取
虽然 SSS 一定可以由本原根生成，但可能存在 ωn′\omega_n'ωn′​ 不是本原 nnn 次根，却依然满足 S=F(ωn′)S = F(\omega_n')S=F(ωn′​)。

实例分析 (n=15n=15n=15)：
考虑 x15−1x^{15}-1x15−1 在 F=F2F = \mathbb{F}_2F=F2​ 上的分裂域。

  1. 多项式 x4+x+1x^4+x+1x4+x+1 在 F2\mathbb{F}_2F2​ 上不可约。设其根为 α\alphaα。
    经计算 ord(α)=15\text{ord}(\alpha) = 15ord(α)=15，即 α\alphaα 为本原根。
    此时 S=F2(α)=F16S = \mathbb{F}_2(\alpha) = \mathbb{F}_{16}S=F2​(α)=F16​。（α\alphaα 也是域 F16\mathbb{F}_{16}F16​ 的本原元）。
  2. 考虑另一个多项式 m3(x)=x4+x3+x2+x+1m_3(x) = x^4+x^3+x^2+x+1m3​(x)=x4+x3+x2+x+1。
    它有一个根为 α3\alpha^3α3。因为 ord(α3)=15(15,3)=5≠15\text{ord}(\alpha^3) = \frac{15}{(15, 3)} = 5 \neq 15ord(α3)=(15,3)15​=5=15，所以 α3\alpha^3α3 不是 151515 次本原根。
  3. 但是，依然有 S=F2(α3)=F16S = \mathbb{F}_2(\alpha^3) = \mathbb{F}_{16}S=F2​(α3)=F16​。
    (这说明即便元素不是本原 nnn 次根，它生成的域也可能等于整个分圆扩张域)。

由于 char(F)∤n\text{char}(F) \nmid nchar(F)∤n，该扩张是可分的，且是分裂域，因此 S/FS/FS/F 是 伽罗瓦扩张 (Galois Extension)。

伽罗瓦群的结构映射

我们要研究伽罗瓦群 G=Gal(S/F)G = \text{Gal}(S/F)G=Gal(S/F) 的结构。
对于任意 σ∈Gal(S/F)\sigma \in \text{Gal}(S/F)σ∈Gal(S/F)，σ\sigmaσ 由它在生成元 ωn\omega_nωn​ 上的作用，即 σ(ωn)\sigma(\omega_n)σ(ωn​) 唯一确定。
由于 σ(ωn)\sigma(\omega_n)σ(ωn​) 必须仍是 xn−1x^n - 1xn−1 的根，且必须保持元素的阶不变（即仍为本原根），故：

σ(ωn)=ωnk,其中 gcd⁡(n,k)=1\sigma(\omega_n) = \omega_n^k, \quad \text{其中 } \gcd(n, k) = 1 σ(ωn​)=ωnk​,其中 gcd(n,k)=1

定理：伽罗瓦群的嵌入
存在一个单射群同态 kkk:

Gal(S/F)↪Zn∗\text{Gal}(S/F) \hookrightarrow \mathbb{Z}_n^* Gal(S/F)↪Zn∗​

这意味着分圆扩张的伽罗瓦群同构于模 nnn 乘法群 Zn∗\mathbb{Z}_n^*Zn∗​ 的一个子群。因此，分圆扩张的伽罗瓦群总是阿贝尔群 (Abelian)。

证明思路：
引入映射 k:σ↦k(σ)k: \sigma \mapsto k(\sigma)k:σ↦k(σ)，使得 k(σ)=log⁡ωnσ(ωn)k(\sigma) = \log_{\omega_n}\sigma(\omega_n)k(σ)=logωn​​σ(ωn​)，即满足 σ(ωn)=ωnk(σ)\sigma(\omega_n) = \omega_n^{k(\sigma)}σ(ωn​)=ωnk(σ)​。

  1. 同态性：σ(τ(ωn))=σ(ωnk(τ))=(ωnk(σ))k(τ)=ωnk(σ)k(τ)\sigma(\tau(\omega_n)) = \sigma(\omega_n^{k(\tau)}) = (\omega_n^{k(\sigma)})^{k(\tau)} = \omega_n^{k(\sigma)k(\tau)}σ(τ(ωn​))=σ(ωnk(τ)​)=(ωnk(σ)​)k(τ)=ωnk(σ)k(τ)​，对应于群乘法。
  2. 单射性：若 k(σ)=1k(\sigma) = 1k(σ)=1，则 σ(ωn)=ωn\sigma(\omega_n) = \omega_nσ(ωn​)=ωn​，即 σ\sigmaσ 固定生成元，故 σ=id\sigma = \text{id}σ=id。

注意：kkk 不一定是满射。

  • 在 F=QF = \mathbb{Q}F=Q 时，kkk 是同构，即 ∣Gal(S/F)∣=ϕ(n)|\text{Gal}(S/F)| = \phi(n)∣Gal(S/F)∣=ϕ(n)。
  • 在有限域 FqF_qFq​ 上，往往只是真子群。
  • 例子：考虑 F=F2F = \mathbb{F}_2F=F2​，扩张 F16=F2(ω15)F_{16} = \mathbb{F}_2(\omega_{15})F16​=F2​(ω15​)。∣Gal∣=4|\text{Gal}| = 4∣Gal∣=4，但 ∣Z15∗∣=ϕ(15)=8|\mathbb{Z}_{15}^*| = \phi(15) = 8∣Z15∗​∣=ϕ(15)=8。

记 UnU_nUn​ 为 xn−1x^n - 1xn−1 的根集。对于任意 ω∈Un\omega \in U_nω∈Un​，ord(ω)\text{ord}(\omega)ord(ω) 必整除 nnn。

可约性讨论：有理域 vs 有限域

    如果 ∣Gal(S/F)∣=ϕ(n)|\text{Gal}(S/F)| = \phi(n)∣Gal(S/F)∣=ϕ(n)，则 deg⁡(min⁡(ωn,F))=ϕ(n)=deg⁡(Qn)\deg(\min(\omega_n, F)) = \phi(n) = \deg(Q_n)deg(min(ωn​,F))=ϕ(n)=deg(Qn​)。

综上所述，我们得到如下判定准则：

判定准则

∣Gal(S/F)∣=ϕ(n)  ⟺  Gal(S/F)≅Zn∗  ⟺  Qn(x) 在 F 上不可约|\text{Gal}(S/F)| = \phi(n) \iff \text{Gal}(S/F) \cong \mathbb{Z}_n^* \iff Q_n(x) \text{ 在 } F \text{ 上不可约} ∣Gal(S/F)∣=ϕ(n)⟺Gal(S/F)≅Zn∗​⟺Qn​(x) 在 F 上不可约

不同情形下的可约性

情形 1：F=QF = \mathbb{Q}F=Q

定理：有理域上的不可约性
Qn(x)Q_n(x)Qn​(x) 在 Q[x]\mathbb{Q}[x]Q[x] 上是不可约的。

  • 扩张次数：[S:Q]=deg⁡(Qn)=ϕ(n)[S : \mathbb{Q}] = \deg(Q_n) = \phi(n)[S:Q]=deg(Qn​)=ϕ(n)。
  • 伽罗瓦群：Gal(S/Q)≅Zn∗\text{Gal}(S/\mathbb{Q}) \cong \mathbb{Z}_n^*Gal(S/Q)≅Zn∗​（满射同构）。

情形 2：F=FqF = \mathbb{F}_qF=Fq​ (有限域)

结论：有限域上的可约性
Qn(x)Q_n(x)Qn​(x) 在有限域上通常是可约的，除非 Gal(S/F)\text{Gal}(S/F)Gal(S/F) 生成了整个 Zn∗\mathbb{Z}_n^*Zn∗​。

[S:F][S : F][S:F] 等于 ωn\omega_nωn​ 在 FFF 上的极小多项式次数，这通常小于 ϕ(n)\phi(n)ϕ(n)。

实例分析

例 1：F2\mathbb{F}_2F2​ 上的 x7−1x^7 - 1x7−1

计算 n=7n=7n=7 的分圆多项式 Q7(x)Q_7(x)Q7​(x)：
在 Q\mathbb{Q}Q 上，Q7(x)=x6+x5+x4+x3+x2+x+1Q_7(x) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1Q7​(x)=x6+x5+x4+x3+x2+x+1（不可约）。
但在 F2\mathbb{F}_2F2​ 上，我们发现 Q7(x)Q_7(x)Q7​(x) 可以分解：

Q7(x)=(x3+x+1)(x3+x2+1)Q_7(x) = (x^3 + x + 1)(x^3 + x^2 + 1) Q7​(x)=(x3+x+1)(x3+x2+1)

这两个因子在 F2\mathbb{F}_2F2​ 上都不可约。
因此，扩张次数 [S:F2]=3[S : \mathbb{F}_2] = 3[S:F2​]=3（而不是 ϕ(7)=6\phi(7)=6ϕ(7)=6）。

例 2：Q(ω7)\mathbb{Q}(\omega_7)Q(ω7​) 的伽罗瓦对应 (Galois Correspondence)

1. 设定与群结构
设 F=QF = \mathbb{Q}F=Q，ω\omegaω 为本原 777 次单位根，E=Q(ω)E = \mathbb{Q}(\omega)E=Q(ω)。
已知扩张次数 [E:Q]=ϕ(7)=6[E:\mathbb{Q}] = \phi(7) = 6[E:Q]=ϕ(7)=6。
伽罗瓦群结构如下：

Gal(E/Q)≅Z7∗={1,2,3,4,5,6}=⟨3⟩\text{Gal}(E/\mathbb{Q}) \cong \mathbb{Z}_7^* = \{1, 2, 3, 4, 5, 6\} = \langle 3 \rangle Gal(E/Q)≅Z7∗​={1,2,3,4,5,6}=⟨3⟩

这是一个 6 阶循环群。记 σk\sigma_kσk​ 为满足 σk(ω)=ωk\sigma_k(\omega) = \omega^kσk​(ω)=ωk 的自同构。
生成元为 σ3\sigma_3σ3​，即 Gal(E/Q)=⟨σ3⟩\text{Gal}(E/\mathbb{Q}) = \langle \sigma_3 \rangleGal(E/Q)=⟨σ3​⟩。

2. 子群格 (Lattice of Subgroups)
⟨σ3⟩\langle \sigma_3 \rangle⟨σ3​⟩ 的子群由其阶数的因子唯一确定。666 的因子有 1,2,3,61, 2, 3, 61,2,3,6。
共有 4 个子群：

  • Order 1: {id}\{ \text{id} \}{id}
  • Order 6: ⟨σ3⟩\langle \sigma_3 \rangle⟨σ3​⟩ (全群)
  • Order 3: ⟨σ32⟩=⟨σ2⟩={1,2,4}\langle \sigma_3^2 \rangle = \langle \sigma_2 \rangle = \{1, 2, 4\}⟨σ32​⟩=⟨σ2​⟩={1,2,4}
  • Order 2: ⟨σ33⟩=⟨σ6⟩={1,6}\langle \sigma_3^3 \rangle = \langle \sigma_6 \rangle = \{1, 6\}⟨σ33​⟩=⟨σ6​⟩={1,6}

3. 寻找中间域：以 ⟨σ33⟩\langle \sigma_3^3 \rangle⟨σ33​⟩ 为例
我们考察阶为 2 的子群 H=⟨σ33⟩H = \langle \sigma_3^3 \rangleH=⟨σ33​⟩。
由于 σ33≡σ27≡σ6(mod7)\sigma_3^3 \equiv \sigma_{27} \equiv \sigma_6 \pmod 7σ33​≡σ27​≡σ6​(mod7)，故 H=⟨σ6⟩H = \langle \sigma_6 \rangleH=⟨σ6​⟩。
计算其固定域 Fix(H)\text{Fix}(H)Fix(H)。

  • 构造固定元素：
    考察 σ6\sigma_6σ6​ 的作用：

    σ6(ω)=ω6\sigma_6(\omega) = \omega^6 σ6​(ω)=ω6

    σ6(ω6)=(ω6)6=ω36=ω1\sigma_6(\omega^6) = (\omega^6)^6 = \omega^{36} = \omega^1 σ6​(ω6)=(ω6)6=ω36=ω1

    显见 σ6\sigma_6σ6​ 交换 ω\omegaω 和 ω6\omega^6ω6。因此，元素 α=ω+ω6\alpha = \omega + \omega^6α=ω+ω6 在 σ6\sigma_6σ6​ 作用下不变：

    σ6(ω+ω6)=ω6+ω\sigma_6(\omega + \omega^6) = \omega^6 + \omega σ6​(ω+ω6)=ω6+ω

    于是 Q(ω+ω6)⊆Fix(H)\mathbb{Q}(\omega + \omega^6) \subseteq \text{Fix}(H)Q(ω+ω6)⊆Fix(H)。

  • 确定扩张次数：
    根据伽罗瓦理论基本定理：

    [E:Fix(H)]=∣H∣=2[E : \text{Fix}(H)] = |H| = 2 [E:Fix(H)]=∣H∣=2

    [Fix(H):Q]=[E:Q]/2=6/2=3[\text{Fix}(H) : \mathbb{Q}] = [E : \mathbb{Q}] / 2 = 6 / 2 = 3 [Fix(H):Q]=[E:Q]/2=6/2=3

    对于我们构造的域 K=Q(ω+ω6)K = \mathbb{Q}(\omega + \omega^6)K=Q(ω+ω6)，有 K⊆Fix(H)K \subseteq \text{Fix}(H)K⊆Fix(H)。
    这意味着 [K:Q][K : \mathbb{Q}][K:Q] 必须整除 [Fix(H):Q]=3[\text{Fix}(H) : \mathbb{Q}] = 3[Fix(H):Q]=3。
    所以 [K:Q][K : \mathbb{Q}][K:Q] 只能是 111 或 333。

  • 排除 [K:Q]=1[K : \mathbb{Q}] = 1[K:Q]=1 (反证法)：
    如果 [K:Q]=1[K : \mathbb{Q}] = 1[K:Q]=1，则 ω+ω6∈Q\omega + \omega^6 \in \mathbb{Q}ω+ω6∈Q。
    这意味着该元素被伽罗瓦群中的所有元素固定。
    我们任取 σ∈Gal(E/Q)\sigma \in \text{Gal}(E/\mathbb{Q})σ∈Gal(E/Q)，例如取 σ2\sigma_2σ2​（注意 σ2(ω)=ω2\sigma_2(\omega)=\omega^2σ2​(ω)=ω2）：
    若 ω+ω6∈Q\omega + \omega^6 \in \mathbb{Q}ω+ω6∈Q，则应有 σ2(ω+ω6)=ω+ω6\sigma_2(\omega + \omega^6) = \omega + \omega^6σ2​(ω+ω6)=ω+ω6。

    计算左边：

    σ2(ω+ω6)=ω2+ω12=ω2+ω5\sigma_2(\omega + \omega^6) = \omega^2 + \omega^{12} = \omega^2 + \omega^5 σ2​(ω+ω6)=ω2+ω12=ω2+ω5

    令其相等：

    ω2+ω5=ω+ω6  ⟹  ω6−ω5−ω2+ω=0\omega^2 + \omega^5 = \omega + \omega^6 \implies \omega^6 - \omega^5 - \omega^2 + \omega = 0 ω2+ω5=ω+ω6⟹ω6−ω5−ω2+ω=0

    这表明 ω\omegaω 是多项式 f(x)=x6−x5−x2+xf(x) = x^6 - x^5 - x^2 + xf(x)=x6−x5−x2+x 的根。

    矛盾：ω\omegaω 在 Q\mathbb{Q}Q 上的极小多项式是分圆多项式 Q7(x)=x6+x5+x4+x3+x2+x+1Q_7(x) = x^6 + x^5 + x^4 + x^3 + x^2 + x + 1Q7​(x)=x6+x5+x4+x3+x2+x+1。
    由于 deg⁡(f(x))<deg⁡(Q7(x))\deg(f(x)) < \deg(Q_7(x))deg(f(x))<deg(Q7​(x)) 且 f(x)≠0f(x) \neq 0f(x)=0，这与极小多项式的定义矛盾。

    ∴ω+ω6∉Q\therefore \omega + \omega^6 \notin \mathbb{Q}∴ω+ω6∈/Q，即 [K:Q]≠1[K : \mathbb{Q}] \neq 1[K:Q]=1。

  • 结论：
    必须有 [Q(ω+ω6):Q]=3[\mathbb{Q}(\omega + \omega^6) : \mathbb{Q}] = 3[Q(ω+ω6):Q]=3。
    因此，中间域正是：

    Fix(⟨σ6⟩)=Q(ω+ω6)=Q(ω+ω−1)\text{Fix}(\langle \sigma_6 \rangle) = \mathbb{Q}(\omega + \omega^6) = \mathbb{Q}(\omega + \omega^{-1}) Fix(⟨σ6​⟩)=Q(ω+ω6)=Q(ω+ω−1)

4. 另一个中间域 (简述)
对于阶为 3 的子群 ⟨σ32⟩=⟨σ2⟩={σ1,σ2,σ4}\langle \sigma_3^2 \rangle = \langle \sigma_2 \rangle = \{ \sigma_1, \sigma_2, \sigma_4 \}⟨σ32​⟩=⟨σ2​⟩={σ1​,σ2​,σ4​}。
考察元素 β=ω+ω2+ω4\beta = \omega + \omega^2 + \omega^4β=ω+ω2+ω4。
验证 σ2(β)\sigma_2(\beta)σ2​(β)：

σ2(ω+ω2+ω4)=ω2+ω4+ω8=ω2+ω4+ω=β\sigma_2(\omega + \omega^2 + \omega^4) = \omega^2 + \omega^4 + \omega^8 = \omega^2 + \omega^4 + \omega = \beta σ2​(ω+ω2+ω4)=ω2+ω4+ω8=ω2+ω4+ω=β

同理可证 Q(β)\mathbb{Q}(\beta)Q(β) 即为对应的 2 次中间域。
