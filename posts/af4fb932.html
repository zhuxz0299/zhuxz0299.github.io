高斯引理 (Gaussian Lemma)

本原多项式 (Primitive Polynomial)

定义 (Content 与本原多项式)

假定 RRR 是一个唯一分解整环 (UFD)。对于 R[x]R[x]R[x] 上的一个多项式 f(x)=∑i=0naixif(x) = \sum_{i=0}^{n} a_i x^{i}f(x)=∑i=0n​ai​xi，其内容 (content) c(f)c(f)c(f) 定义为其所有系数的最大公约数，即 c(f)=defgcd⁡(a0,a1,…,an)c(f) \overset{\underset{\mathrm{def}}{}}{=} \operatorname{gcd}(a_0, a_1, \ldots ,a_n)c(f)=def​gcd(a0​,a1​,…,an​)。

如果 1∈c(f)1 \in c(f)1∈c(f)，我们称 fff 是本原的 (primitive)。

注： c(f)c(f)c(f) 是一个由 gcd⁡\operatorname{gcd}gcd 及其所有相伴元构成的集合。1∈c(f)1 \in c(f)1∈c(f) 意味着 c(f)c(f)c(f) 是 RRR 中所有单位元的集合，即 fff 的所有系数没有共同的不可约因子。

本原多项式的乘积

性质： 设 RRR 是 UFD。若 f,g∈R[x]f, g \in R[x]f,g∈R[x] 都是本原多项式，则其乘积 fgfgfg 也是本原多项式。

证明

我们使用反证法。假设 fgfgfg 不是本原的。

  1. 如果 fgfgfg 不是本原的，那么 c(fg)c(fg)c(fg) 不是 RRR 中的单位元。因此，存在 RRR 中的一个不可约元 ppp 使得 p∣c(fg)p \mid c(fg)p∣c(fg)。这意味着 ppp 整除 fgfgfg 的所有系数。
  2. 考虑环 RRR 上的自然同态 ϕ:R→R/⟨p⟩\phi: R \to R/\langle p \rangleϕ:R→R/⟨p⟩。由于 RRR 是 UFD 且 ppp 是不可约元， ⟨p⟩\langle p \rangle⟨p⟩ 是一个素理想，因此 R/⟨p⟩R/\langle p \rangleR/⟨p⟩ 是一个整环。
  3. 我们将此同态扩展到多项式环 ϕˉ:R[x]→(R/⟨p⟩)[x]\bar{\phi}: R[x] \to (R/\langle p \rangle)[x]ϕˉ​:R[x]→(R/⟨p⟩)[x]，其定义为 ϕˉ(∑aixi)=∑ϕ(ai)xi\bar{\phi}(\sum a_i x^i) = \sum \phi(a_i) x^iϕˉ​(∑ai​xi)=∑ϕ(ai​)xi。
  4. 令 fˉ=ϕˉ(f)\bar{f} = \bar{\phi}(f)fˉ​=ϕˉ​(f)，gˉ=ϕˉ(g)\bar{g} = \bar{\phi}(g)gˉ​=ϕˉ​(g)。
  5. 由于 fff 是本原的， ppp 不能整除 fff 的所有系数，所以 fˉ≠0\bar{f} \neq 0fˉ​=0。同理， ggg 也是本原的，所以 gˉ≠0\bar{g} \neq 0gˉ​=0。
  6. 由于 ppp 整除 fgfgfg 的所有系数，我们有 ϕˉ(fg)=fˉgˉ=0\bar{\phi}(fg) = \bar{f}\bar{g} = 0ϕˉ​(fg)=fˉ​gˉ​=0。
  7. 然而，因为 R/⟨p⟩R/\langle p \rangleR/⟨p⟩ 是一个整环，多项式环 (R/⟨p⟩)[x](R/\langle p \rangle)[x](R/⟨p⟩)[x] 也是一个整环。在整环中，fˉ≠0\bar{f} \neq 0fˉ​=0 且 gˉ≠0\bar{g} \neq 0gˉ​=0 蕴涵着 fˉgˉ≠0\bar{f}\bar{g} \neq 0fˉ​gˉ​=0。
  8. 这与第 6 步的 fˉgˉ=0\bar{f}\bar{g} = 0fˉ​gˉ​=0 产生矛盾。
  9. 因此，最初的假设（fgfgfg 不是本原的）是错误的。fgfgfg 必定是本原多项式。

分式域 (Field of Fractions)

定义 (分式域)

设 RRR 是一个 UFD，FFF 是 RRR 的分式域，定义为：

F=def{ab∣a,b∈R,b≠0}F \overset{\underset{\mathrm{def}}{}}{=} \left\{ \frac{a}{b} \mid a,b \in R, b\neq 0 \right\} F=def​{ba​∣a,b∈R,b=0}

在 FFF 上定义加法和乘法：

ab+cd=ad+bcdbab⋅cd=acbd\frac{a}{b} + \frac{c}{d} = \frac{ad+bc}{db} \quad \frac{a}{b} \cdot \frac{c}{d} = \frac{ac}{bd} ba​+dc​=dbad+bc​ba​⋅dc​=bdac​

可以验证 (F,+,⋅)(F, +, \cdot)(F,+,⋅) 构成一个域。FFF 是包含 RRR 的最小域。

高斯引理 (Gaussian Lemma)

定理 (高斯引理)：

设 RRR 是一个 UFD，FFF 是 RRR 的分式域。对于一个本原多项式 f∈R[x]f \in R[x]f∈R[x] 且 deg⁡(f)≥1\deg(f) \ge 1deg(f)≥1，则：

fff 在 R[x]R[x]R[x] 中可约   ⟺  f\iff f⟺f 在 F[x]F[x]F[x] 中可约。

例子

如果 f∈Z[x]f \in \mathbb{Z}[x]f∈Z[x]，fff 在 Z[x]\mathbb{Z}[x]Z[x] 中不可约   ⟺  f\iff f⟺f 在 Q[x]\mathbb{Q}[x]Q[x] 中不可约。

爱森斯坦判别法 (Eisenstein’s Criterion)

高斯引理将域上的多项式的可约性缩小到环；而此判别法用于判断一个多项式在环上是否可约。

定理叙述

定理 (Eisenstein 判别法)：

设 RRR 是 UFD，f(x)=∑i=0naixi∈R[x]f(x) = \sum_{i=0}^{n} a_i x^i \in R[x]f(x)=∑i=0n​ai​xi∈R[x] 是一个本原多项式，且 deg⁡(f)≥1\deg(f) \ge 1deg(f)≥1。

如果存在 RRR 中的一个不可约元 ppp 满足以下所有条件：

  1. p∤anp \nmid a_np∤an​ （ppp 不整除首项系数）
  2. p∣aip \mid a_ip∣ai​ （对于所有 0≤i≤n−10 \le i \le n-10≤i≤n−1，ppp 整除所有其他系数）
  3. p2∤a0p^2 \nmid a_0p2∤a0​ （ppp 的平方不整除常数项）

则 f(x)f(x)f(x) 是 R[x]R[x]R[x] 中的不可约元。

证明

我们使用反证法。假设 f(x)f(x)f(x) 在 R[x]R[x]R[x] 中可约，即 f(x)=g(x)h(x)f(x) = g(x) h(x)f(x)=g(x)h(x)，其中 g(x),h(x)∈R[x]g(x), h(x) \in R[x]g(x),h(x)∈R[x] 且 deg⁡(g)≥1,deg⁡(h)≥1\deg(g) \ge 1, \deg(h) \ge 1deg(g)≥1,deg(h)≥1。

  1. 设 g(x)=∑i=0rbixig(x) = \sum_{i=0}^{r} b_i x^ig(x)=∑i=0r​bi​xi 且 h(x)=∑j=0scjxjh(x) = \sum_{j=0}^{s} c_j x^jh(x)=∑j=0s​cj​xj，其中 r≥1,s≥1,r+s=nr \ge 1, s \ge 1, r+s = nr≥1,s≥1,r+s=n。
  2. 考察常数项：a0=b0c0a_0 = b_0 c_0a0​=b0​c0​。
    • 根据条件 (2) 和 (3)，我们知道 p∣a0p \mid a_0p∣a0​ 但 p2∤a0p^2 \nmid a_0p2∤a0​。
    • 这意味着 ppp 必须整除 b0b_0b0​ 和 c0c_0c0​ 中的恰好一个。
    • 不失一般性，我们假设 p∣b0p \mid b_0p∣b0​ 且 p∤c0p \nmid c_0p∤c0​。
  3. 考察首项系数：an=brcsa_n = b_r c_san​=br​cs​。
    • 根据条件 (1)，p∤anp \nmid a_np∤an​。
    • 这意味着 p∤brp \nmid b_rp∤br​ 且 p∤csp \nmid c_sp∤cs​。
  4. 现在我们来分析 g(x)g(x)g(x) 的系数 b0,b1,…,brb_0, b_1, \ldots, b_rb0​,b1​,…,br​。
    • 我们知道 p∣b0p \mid b_0p∣b0​ (来自第 2 步) 且 p∤brp \nmid b_rp∤br​ (来自第 3 步)。
    • 因此，必然存在一个最小的下标 kkk ( 1≤k≤r1 \le k \le r1≤k≤r ) 使得 p∤bkp \nmid b_kp∤bk​ 但 ppp 整除 kkk 之前的所有 bib_ibi​ (即 p∣b0,p∣b1,…,p∣bk−1p \mid b_0, p \mid b_1, \ldots, p \mid b_{k-1}p∣b0​,p∣b1​,…,p∣bk−1​)。
  5. 考察 f(x)f(x)f(x) 的第 kkk 项系数 aka_kak​：

    ak=bkc0+bk−1c1+⋯+b1ck−1+b0cka_k = b_k c_0 + b_{k-1} c_1 + \cdots + b_1 c_{k-1} + b_0 c_k ak​=bk​c0​+bk−1​c1​+⋯+b1​ck−1​+b0​ck​

  6. 分析 aka_kak​：
    • 由于 k≤r<r+s=nk \le r < r+s = nk≤r<r+s=n，根据定理条件 (2)，我们有 p∣akp \mid a_kp∣ak​。

  7. 推导矛盾：

域的扩张 (Field Extensions)

基本定义

  7. 综上，BBB 是 K/FK/FK/F 的一组基，其元素个数为 m⋅nm \cdot nm⋅n。故 [K:F]=mn=[K:E]⋅[E:F][K:F] = mn = [K:E] \cdot [E:F][K:F]=mn=[K:E]⋅[E:F]。

推广：

对于 F1<F2<⋯<FnF_1 < F_2 < \cdots <F_nF1​<F2​<⋯<Fn​，有 [Fn:F1]=[Fn:Fn−1]⋯[F2:F1][F_n: F_1] = [F_n: F_{n-1}] \cdots [F_2:F_1][Fn​:F1​]=[Fn​:Fn−1​]⋯[F2​:F1​]。

有限域 (Finite Fields)

性质：

F<EF<EF<E，如果 [E:F]=n[E:F] = n[E:F]=n，那么 EEE 中的每个元素都可以唯一地表示为 FFF 中 nnn 个元素的线性组合。

如果 FFF 是有限域，则 ∣E∣=∣F∣n|E| = |F|^n∣E∣=∣F∣n。

推论：

设 FFF 是一个有限域。

  1. FFF 的特征 char(F)char(F)char(F) 必定为素数 ppp。
  2. FFF 包含一个同构于 Zp\mathbb{Z}_pZp​ 的子域（素域）。
  3. FFF 可以看作是 Zp\mathbb{Z}_pZp​ 的扩张。设 [F:Zp]=n[F: \mathbb{Z}_p] = n[F:Zp​]=n。
  4. 则 ∣F∣=∣Zp∣n=pn|F| = |\mathbb{Z}_p|^n = p^n∣F∣=∣Zp​∣n=pn。

问题： ∀n>0\forall n>0∀n>0，是否都存在一个 pnp^npn 元域？（答案：是的，这种域称为伽罗瓦域 GF(pn)GF(p^n)GF(pn)，且在同构意义下是唯一的。）

域的生成元 (Generators of a Field)

考虑 F<E,X⊆EF<E, X \subseteq EF<E,X⊆E。

定义

定义 (生成的环)

EEE 中所有包含 FFF 和 XXX 的环的交集（即包含 FFF 和 XXX 的最小环）称为 XXX 在 FFF 上生成的环，记为 F[X]F[X]F[X]。

  • 如果 X={α}X = \{ \alpha \}X={α}，记为 F[α]F[\alpha]F[α]。
  • 如果 X={α1,…,αn}X = \{ \alpha_1, \ldots ,\alpha_n \}X={α1​,…,αn​}，记为 F[α1,…αn]F[\alpha_1, \ldots \alpha_n]F[α1​,…αn​]。

EEE 中所有包含 FFF 和 XXX 的域的交集（即包含 FFF 和 XXX 的最小域）称为 XXX 在 FFF 上生成的域，记为 F(X)F(X)F(X)。

  • 如果 X={α}X = \{ \alpha \}X={α}，记为 F(α)F(\alpha)F(α)。
  • 如果 X={α1,…,αn}X = \{ \alpha_1, \ldots ,\alpha_n \}X={α1​,…,αn​}，记为 F(α1,…αn)F(\alpha_1, \ldots \alpha_n)F(α1​,…αn​)。

元素表示

单元素生成

  • F[α]={f(α)∣f∈F[x]}F[\alpha] = \{ f(\alpha) \mid f \in F[x] \}F[α]={f(α)∣f∈F[x]}
    • 说明： 集合 {f(α)∣f∈F[x]}\{ f(\alpha) \mid f \in F[x] \}{f(α)∣f∈F[x]} 是一个包含 FFF 和 α\alphaα 的环，而 F[α]F[\alpha]F[α] 是包含 FFF 和 α\alphaα 的最小环，所以 F[α]⊆{f(α)∣…}F[\alpha] \subseteq \{ f(\alpha) \mid \ldots \}F[α]⊆{f(α)∣…}。反之， F[α]F[\alpha]F[α] 必须包含 FFF 和 α\alphaα 及其所有乘积和加和，即必须包含所有 f(α)f(\alpha)f(α)，故两者相等。
  • F(α)={f(α)g(α)∣f,g∈F[x],g(α)≠0}F(\alpha) = \left\{ \frac{f(\alpha)}{g(\alpha)} \mid f, g \in F[x], g(\alpha)\neq 0 \right\}F(α)={g(α)f(α)​∣f,g∈F[x],g(α)=0}

代数扩张 (Algebraic Extensions)

代数元与超越元

定义 (代数/超越扩张)

F<EF < EF<E 称为是代数的 (algebraic)，如果 ∀α∈E\forall \alpha \in E∀α∈E， α\alphaα 在 FFF 上是代数元（即 α\alphaα 是 F[x]F[x]F[x] 中某个非零多项式的根）。

否则，如果 EEE 中存在 FFF 上的超越元， F<EF<EF<E 称为是超越的 (transcendental)。

注： 我们后续基本都讨论代数扩张。

单代数扩张 (Simple Algebraic Extension)

定义 (单代数扩张)

设 F<EF < EF<E 是代数扩张，α∈E\alpha \in Eα∈E。F<F(α)F < F(\alpha)F<F(α) 称为单代数扩张。

重要性质： 如果 α\alphaα 是 FFF 上的代数元，则 F(α)=F[α]F(\alpha) = F[\alpha]F(α)=F[α]。

证明

  1. F[α]⊆F(α)F[\alpha] \subseteq F(\alpha)F[α]⊆F(α) 是显然的（域总是环）。
  2. 我们只需证明 F(α)⊆F[α]F(\alpha) \subseteq F[\alpha]F(α)⊆F[α]。
  3. 取 F(α)F(\alpha)F(α) 中的任意元素 f(α)g(α)\frac{f(\alpha)}{g(\alpha)}g(α)f(α)​，其中 g(α)≠0g(\alpha) \neq 0g(α)=0。
  4. 由于 α\alphaα 是代数元，存在 FFF 上唯一的首一不可约多项式 Pα(x)P_{\alpha}(x)Pα​(x)（称为 α\alphaα 的极小多项式），使得 Pα(α)=0P_{\alpha}(\alpha)=0Pα​(α)=0。
  5. 因为 g(α)≠0g(\alpha) \neq 0g(α)=0， Pα(x)P_{\alpha}(x)Pα​(x) 不整除 g(x)g(x)g(x)。
  6. 由于 Pα(x)P_{\alpha}(x)Pα​(x) 在 F[x]F[x]F[x] 中不可约， Pα(x)P_{\alpha}(x)Pα​(x) 和 g(x)g(x)g(x) 互素， gcd⁡(Pα,g)=1\operatorname{gcd}(P_{\alpha}, g) = 1gcd(Pα​,g)=1。
  7. 根据贝祖定理（在 F[x]F[x]F[x] 这个主理想整环上成立），存在 a(x),b(x)∈F[x]a(x), b(x) \in F[x]a(x),b(x)∈F[x] 使得 Pα(x)a(x)+g(x)b(x)=1P_{\alpha}(x) a(x) + g(x) b(x) = 1Pα​(x)a(x)+g(x)b(x)=1。
  8. 将 α\alphaα 代入上式： Pα(α)a(α)+g(α)b(α)=1P_{\alpha}(\alpha) a(\alpha) + g(\alpha) b(\alpha) = 1Pα​(α)a(α)+g(α)b(α)=1。
  9. 由于 Pα(α)=0P_{\alpha}(\alpha) = 0Pα​(α)=0，上式变为 g(α)b(α)=1g(\alpha) b(\alpha) = 1g(α)b(α)=1，即 b(α)=1g(α)b(\alpha) = \frac{1}{g(\alpha)}b(α)=g(α)1​。
  10. 因此， f(α)g(α)=f(α)b(α)\frac{f(\alpha)}{g(\alpha)} = f(\alpha) b(\alpha)g(α)f(α)​=f(α)b(α)。由于 f(x)b(x)f(x)b(x)f(x)b(x) 仍然是 F[x]F[x]F[x] 中的多项式， f(α)b(α)∈F[α]f(\alpha) b(\alpha) \in F[\alpha]f(α)b(α)∈F[α]。
  11. 故 F(α)⊆F[α]F(\alpha) \subseteq F[\alpha]F(α)⊆F[α]。
  12. 综上所述，F(α)=F[α]F(\alpha) = F[\alpha]F(α)=F[α]。

单代数扩张的结构

性质：

设 α\alphaα 在 FFF 上代数，其极小多项式为 Pα(x)P_{\alpha}(x)Pα​(x)。则：

F(α)=F[α]={f(α)∣f∈F[x] 且 deg⁡(f)<deg⁡(Pα)}F(\alpha) = F[\alpha] = \{ f(\alpha) \mid f \in F[x] \text{ 且 } \deg(f) < \deg(P_{\alpha}) \} F(α)=F[α]={f(α)∣f∈F[x] 且 deg(f)<deg(Pα​)}

证明

  1. 取 F[α]F[\alpha]F[α] 中任意元素 h(α)h(\alpha)h(α) (其中 h∈F[x]h \in F[x]h∈F[x])。
  2. 对 h(x)h(x)h(x) 和 Pα(x)P_{\alpha}(x)Pα​(x) 使用多项式除法（带余除法）：
    h(x)=q(x)Pα(x)+r(x)h(x) = q(x) P_{\alpha}(x) + r(x)h(x)=q(x)Pα​(x)+r(x)，其中 r(x)=0r(x) = 0r(x)=0 或 deg⁡(r)<deg⁡(Pα)\deg(r) < \deg(P_{\alpha})deg(r)<deg(Pα​)。
  3. 将 α\alphaα 代入： h(α)=q(α)Pα(α)+r(α)h(\alpha) = q(\alpha) P_{\alpha}(\alpha) + r(\alpha)h(α)=q(α)Pα​(α)+r(α)。
  4. 由于 Pα(α)=0P_{\alpha}(\alpha) = 0Pα​(α)=0， h(α)=r(α)h(\alpha) = r(\alpha)h(α)=r(α)。
  5. 这表明 F[α]F[\alpha]F[α] 中的任何元素都可以用一个次数小于 deg⁡(Pα)\deg(P_{\alpha})deg(Pα​) 的多项式在 α\alphaα 处的值来表示。
  6. 并且，如果 r1(α)=r2(α)r_1(\alpha) = r_2(\alpha)r1​(α)=r2​(α) 且 deg⁡(r1),deg⁡(r2)<deg⁡(Pα)\deg(r_1), \deg(r_2) < \deg(P_{\alpha})deg(r1​),deg(r2​)<deg(Pα​)，则 (r1−r2)(α)=0(r_1-r_2)(\alpha) = 0(r1​−r2​)(α)=0。这要求 Pα(x)∣(r1(x)−r2(x))P_{\alpha}(x) \mid (r_1(x) - r_2(x))Pα​(x)∣(r1​(x)−r2​(x))。但 Pα(x)P_{\alpha}(x)Pα​(x) 的次数更高，这只可能在 r1(x)−r2(x)=0r_1(x) - r_2(x) = 0r1​(x)−r2​(x)=0 时成立，即 r1(x)=r2(x)r_1(x) = r_2(x)r1​(x)=r2​(x)。
  7. 因此，这种表示是唯一的。

推论 (扩度与基)

设 d=deg⁡(Pα)d = \deg(P_{\alpha})d=deg(Pα​)。

  1. F(α)F(\alpha)F(α) 作为 FFF 上线性空间的一组基为 {1,α,…,αd−1}\{ 1, \alpha, \ldots, \alpha^{d-1} \}{1,α,…,αd−1}。
  2. [F(α):F]=d[F(\alpha): F] = d[F(α):F]=d。
  3. （若 FFF 为有限域） ∣F(α)∣=∣F∣d|F(\alpha)| = |F|^d∣F(α)∣=∣F∣d。

同构关系

定理： F[x]/⟨Pα(x)⟩≅F(α)F[x] / \langle P_{\alpha}(x) \rangle \cong F(\alpha)F[x]/⟨Pα​(x)⟩≅F(α)

证明概要

  1. 定义一个映射 φ:F[x]→F(α)\varphi: F[x] \to F(\alpha)φ:F[x]→F(α)， f(x)↦f(α)f(x) \mapsto f(\alpha)f(x)↦f(α)。
  2. 可以验证 φ\varphiφ 是一个环同态。
  3. 根据 F[α]={f(α)∣…}F[\alpha] = \{ f(\alpha) \mid \ldots \}F[α]={f(α)∣…} 的定义， φ\varphiφ 是满同态。
  4. 考察其核 ker(φ)ker(\varphi)ker(φ)：
    ker(φ)={f(x)∈F[x]∣f(α)=0}ker(\varphi) = \{ f(x) \in F[x] \mid f(\alpha) = 0 \}ker(φ)={f(x)∈F[x]∣f(α)=0}。
  5. 根据极小多项式的定义， f(α)=0  ⟺  Pα(x)∣f(x)f(\alpha)=0 \iff P_{\alpha}(x) \mid f(x)f(α)=0⟺Pα​(x)∣f(x)。
  6. 因此 ker(φ)={f(x)∈F[x]∣Pα(x)∣f(x)}=⟨Pα(x)⟩ker(\varphi) = \{ f(x) \in F[x] \mid P_{\alpha}(x) \mid f(x) \} = \langle P_{\alpha}(x) \rangleker(φ)={f(x)∈F[x]∣Pα​(x)∣f(x)}=⟨Pα​(x)⟩（由 Pα(x)P_{\alpha}(x)Pα​(x) 生成的主理想）。
  7. 根据环同构基本定理，F[x]/ker(φ)≅Im⁡(φ)F[x] / ker(\varphi) \cong \operatorname{Im}(\varphi)F[x]/ker(φ)≅Im(φ)。
  8. 即 F[x]/⟨Pα(x)⟩≅F(α)F[x] / \langle P_{\alpha}(x) \rangle \cong F(\alpha)F[x]/⟨Pα​(x)⟩≅F(α)。

性质：

如果 α,β∈E\alpha, \beta \in Eα,β∈E 在 FFF 上是共轭的（即它们有相同的极小多项式 P(x)P(x)P(x)），则 F(α)≅F(β)F(\alpha) \cong F(\beta)F(α)≅F(β)。

简要说明

  • 根据上述定理， F(α)≅F[x]/⟨P(x)⟩F(\alpha) \cong F[x] / \langle P(x) \rangleF(α)≅F[x]/⟨P(x)⟩。
  • 同理， F(β)≅F[x]/⟨P(x)⟩F(\beta) \cong F[x] / \langle P(x) \rangleF(β)≅F[x]/⟨P(x)⟩。
  • 因此， F(α)≅F(β)F(\alpha) \cong F(\beta)F(α)≅F(β)。

  • 因此 [Q(2):Q]=2[\mathbb{Q}(\sqrt{2}): \mathbb{Q}] = 2[Q(2​):Q]=2。
