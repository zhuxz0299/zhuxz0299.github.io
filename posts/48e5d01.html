Windows 软件记录

发表于2026-01-22|更新于2026-01-24|System & HardwareWindows
|总字数:921|阅读时长:3分钟

  • Capslock+

    • 博客快捷使用方向键中有记录。

  • im-select

    • 为了方便在使用 VSCodeVim 插件时，进入 NORMAL 模式时能够自动切换英文输入法的辅助工具。

  • iperf

    • 一个用于测量两台机器之间带宽与延迟的工具。

  • Openlist

    • 博客 WebDAV 协议与网盘的本地挂载中有记录

  • SumatraPDF

    • 一个 PDF 阅读器，足够轻量且支持对文档进行简单的反色。

  • TboxWebdav

    • 让用户可以通过 WebDAV 协议访问新交大云盘。

  • v2rayN

    • 一个图形化的代理工具，支持 Xray，sing-box 等内核。

  • ventory

    • 制作可启动U盘的开源工具，支持各种操作系统。

  • 7-zip

    • 非常好用的压缩/解压缩工具。

  • Adobe Premiere Pro

    • 专业的视频剪辑软件。

  • AMD Software

    • AMD 显卡驱动与管理软件。

  • Apollo

    • 远程串流软件，是 Sunshine 的 Fork，增加了更多的高级功能。

  • Application Verifier

    • 微软官方的应用 BUG 监测工具。不知道哪来的，可能是下载 Visual Studio 附带的。

  • AutoHotkey

    • 在 Windows 上用来给各种组合键自定义脚本，是非常好用的工具。

  • Autorun Organizer 下载地址（找不到官网）

    • 用于设置 Windows 的开机自启动项，可以禁用某些自启动，也可以设置开机之后一定时间再自启动，缓解刚开机是大量应用同时启动导致的卡顿。

  • BetterGI

    • 原神脚本软件。

  • Cherry Studio

    • 提供和 AI 交互的 GUI。

  • Clash Verge

    • 非常强大的代理软件。

  • DAUM PotPlayer

    • 开箱即用且功能丰富的视频播放器。

  • Docker-desktop

    • 目前想在 Windows 上用 Docker 不得不品的一环。

  • dotnet

    • .NET 开发环境

  • eudic

    • 欧陆词典，目前用下来比较好用且美观的本地词典。

  • Git

    • git 工具

  • Google Chrome

    • Chrome 浏览器

  • HandBrake

    • 开源视频格式转换工具

  • HP Print and Scan Doctor

    • 惠普打印机诊断工具。

  • JetBrains

    • JetBrains 全家桶，强大的 IDE 集合。

  • LocalSend

    • 开源跨平台局域网文件传输工具。

  • Logitech G Hub

    • 罗技游戏外设管理驱动。

  • Logi Options+

    • 罗技办公外设管理驱动。

  • Mechrevo Control Center

    • 机械革命电脑控制中心。

  • Microsoft Office

    • 微软办公套件。

  • Microsoft OfficePLUS

    • 微软官方的 Office 插件，提供模板和素材。

  • Microsoft OneDrive

    • 微软推出的云存储服务。

  • Microsoft SQL Server

    • 微软推出的关系型数据库管理系统。

  • Microsoft Visual Studio

    • 微软推出的集成开发环境（IDE）。

  • MiHoYo Launcher

    • 米哈游启动器，用于启动原神、崩坏：星穹铁道等游戏。

  • Moonlight

    • 基于 NVIDIA GameStream 的开源游戏串流客户端。

  • Mozilla Firefox

    • 优秀的开源浏览器。

  • Mozilla Thunderbird

    • Mozilla 出品的开源邮件客户端。

  • Node.js

    • JavaScript 运行时环境。

  • OBS Studio

    • 强大的开源录屏与直播软件。

  • OPPO Connect

    • OPPO 跨屏互联，手机与电脑协同工具。

  • PowerToys

    • 微软官方出品的系统增强工具集，非常实用。

  • qBittorrent

    • 开源 BitTorrent 客户端。

  • RaiDrive

    • 将网盘挂载为本地磁盘的工具。

  • Sangfor EasyConnect

    • 深信服 EasyConnect，用于远程接入办公网络。在电脑上实际上大概是交大 VPN 用到的 64 位的库。

  • SimpleTex

    • 强大的数学公式识别截图 OCR 工具，不过目前来看不如直接用 AI 识别了。

  • Tailscale

    • 基于 WireGuard 的零配置虚拟局域网组网工具。

  • Tencent QQ

    • 腾讯即时通讯软件。

  • WeChat

    • 微信电脑版。

  • Tencent Meeting

    • 腾讯会议。

  • ToDesk

    • 远程桌面控制软件。

  • Typora

    • 优雅的 Markdown 编辑器（所见即所得）。

  • ViGEmBus

    • 虚拟游戏手柄驱动，常配合手柄映射工具使用。不知道哪来的，可能跟着 Apollo 一起装的？

  • Zotero

    • 开源文献管理工具。
