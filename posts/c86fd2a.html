Logic Behind Natural Language Proof

Example 1.

  • Assumptions: AAA is a set, R⊆A×AR \subseteq A\times AR⊆A×A is a binary relation, …
  • Proof step: In order to prove that RRR is an equivalence relation on AAA, we prove that kkk is reflexive on AAA, symmetric and transitive.

Syntax of First Order Logic

只是用原子命题和逻辑连接词，表达能力不够。

Example

  • Commutative law of addition: for any natural number xxx and yyy, x+y=y+xx+y = y+xx+y=y+x
  • 3+5=5+33+5 = 5+33+5=5+3

这两个命题之间的关系无法由原子命题体现。

Suppose

  • P(t1,t2,t3,t4)P(t_1, t_2, t_3, t_4)P(t1​,t2​,t3​,t4​) means (informally) t1+t2=t3+t4t_1+t_2 = t_3+t_4t1​+t2​=t3​+t4​
  • cthreec_{\text{three}}cthree​ is a constant symbol representing 333
  • cfivec_{\text{five}}cfive​ is a constant symbol representing 555

Then

  • Commutative law of addition: ∀x,∀y,P(x,y,y,x)\forall x, \forall y, P(x, y, y, x)∀x,∀y,P(x,y,y,x)
  • 3+5=5+33+5 = 5+33+5=5+3: P(cthree,cfive,cfive,cthree)P(c_{\text{three}}, c_{\text{five}}, c_{\text{five}}, c_{\text{three}})P(cthree​,cfive​,cfive​,cthree​)

Example
How to formulate the following sentence in a logic expression?
The cube of cube of 323232 is a number ends in 222.

Strategy #1:

  • P1(t1,t2)P_1(t_1,t_2)P1​(t1​,t2​): the cube of t1t_1t1​ ends in t2t_2t2​.
  • ccc_ccc​: the cube of 323232.
  • P1(c,ctwo)P_1(c,c_{two})P1​(c,ctwo​): The cube of cube of 323232 is a number ends in 222.

Strategy #2:

  • P2(t1,t2)P_2(t_1,t_2)P2​(t1​,t2​): the cube of cube of t1t_1t1​ ends in t2t_2t2​.
  • cthirty−twoc_{thirty-two}cthirty−two​: 323232.
  • P2(cthirty−two,ctwo)P_2(c_{thirty-two},c_{two})P2​(cthirty−two​,ctwo​): The cube of cube of 323232 is a number ends in 222.

在两个不同的 Strategy 中，对于同一个结论：“The cube of cube of 323232 is a number ends in 222.”，可以做出两个推论：

  • ∃x,P1(x,ctwo)\exists x, P_1(x, c_{\text{two}})∃x,P1​(x,ctwo​)
  • ∃x,P2(x,ctwo)\exists x, P_2(x, c_{\text{two}})∃x,P2​(x,ctwo​)

但是想导出这两个推论，有需要对逻辑结构做不同的解读，这说明上面使用的逻辑语言的表达能力还不够。因此不仅要引入谓词符号，还需要引入函数符号。

  • P3(t1,t2)P_3(t_1,t_2)P3​(t1​,t2​): t1t_1t1​ ends in t2t_2t2​.
  • f(t)f(t)f(t): the cube of ttt
  • cthirty–twoc_{\text{thirty--two}}cthirty–two​: 323232.
  • P3(f(cthirty–two),ctwo)P_3(f(c_{\text{thirty--two}}),c_{\text{two}})P3​(f(cthirty–two​),ctwo​): The cube of cube of 323232 is a number ends in 222.

那么此时就可以说：

  • ∃x,P3(f(x),ctwo)\exists x, P_3(f(x),c_\text{two})∃x,P3​(f(x),ctwo​)
  • ∃x,P3(f(f(x)),ctwo)\exists x, P_3(f(f(x)), c_\text{two})∃x,P3​(f(f(x)),ctwo​)

  • Proposition (ϕ\phiϕ): P(t),Q(t1,t2),ϕ1∧ϕ2,¬ϕ,∃xϕ,∀yψP(t), Q(t_1, t_2), \phi_1 \land \phi_2, \lnot \phi, \exists x\phi, \forall y \psiP(t),Q(t1​,t2​),ϕ1​∧ϕ2​,¬ϕ,∃xϕ,∀yψ

Predicate symbols, function symbols and constants forms the symbol set of a first order language.

The semantics of first order language

命题逻辑在给定真值指派之后就能得到真值，但是一阶逻辑就更加复杂。比如在使用量词的时候，量词的范围会影响真值。

Definition 7 (SSS-structures).
Given a symbol set SSS, an SSS-structure A=(A,α)\mathcal{A} = (A, \alpha)A=(A,α) contains

  • a domain(论域) AAA, which is a nonempty set
  • an interpretation of every predicate symbol, e.g. if PPP is a symbol of binary predicate, then α(P)\alpha(P)α(P) is a mapping from A×A→{T,F}A\times A \to \{ T, F \}A×A→{T,F}
  • an interpretation of every function symbol, e.g. if fff is a symbol of unary function, then α(f)\alpha(f)α(f) is a mapping from A→AA \to AA→A
  • an interpretation of every constant symbol, e.g. if ccc is a constant symbol, then α(c)\alpha(c)α(c) is an element in AAA

上面四条，第一条用 AAA 指定论域；后面三条用 α\alphaα 解释一阶逻辑的符号（谓词符号，函数符号，常量符号），有了 α\alphaα 之后，这些符号的意义才确定下来（就像命题逻辑中，ppp只是一个符号，只有给定了真值指派之后，真值才确定下来）。SSS 指的就是一阶逻辑的符号集。
