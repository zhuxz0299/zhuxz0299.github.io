Sub-classes of Propositions

Definition 1. A tautology (重言式, 永真式) is a compound proposition that is always true, no matter what the truth values of the propositional variables are. In other words, phiphiphi is a tautology if ⟦ϕ⟧J=T\llbracket \phi \rrbracket_{\mathcal{J}} = T[[ϕ]]J​=T for any truth assignment J\mathcal{J}J.

Definition 2. A contradiction (矛盾式) is a compound proposition that is always false, no matter what the truth values of the propositional variables are. In other words, phiphiphi is a contradiction if ⟦ϕ⟧J=F\llbracket \phi \rrbracket_{\mathcal{J}}=F[[ϕ]]J​=F for any truth assignment J\mathcal{J}J.

Definition 3. A compound proposition is satisfiable (可满足的) if it is true under some truth assignment. In other words, phiphiphi is satisfiable if there exists a truth assignment J\mathcal{J}J such that ⟦ϕ⟧J=T\llbracket \phi \rrbracket_{\mathcal{J}}=T[[ϕ]]J​=T

Example 4. p∨¬pp \vee \neg pp∨¬p is a tautology, and is satisfiable.

Example 5. p∧¬pp \wedge \neg pp∧¬p is a contradiction, and is not satisfiable.

Example 6. p∧qp \wedge qp∧q is not a tautology, is not a contradiction, and is satisfiable.

Theorem 7.

  • ϕ∨¬ϕ\phi \lor \neg\phiϕ∨¬ϕ is a tautology.
  • ϕ∧¬ϕ\phi \land \neg\phiϕ∧¬ϕ is a contradiction.

Theorem 8.

  • ϕ\phiϕ is a tautology if and only if ¬ϕ\neg\phi¬ϕ is a contradiction.
  • ϕ\phiϕ is a contradiction if and only if ¬ϕ\neg\phi¬ϕ is a tautology.
  • ϕ\phiϕ is satisfiable if and only if ϕ\phiϕ is not a contradiction.
  • ϕ\phiϕ is a tautology if and only if ¬ϕ\neg\phi¬ϕ is not satisfiable.

Theorem 9.

  • ϕ1,ϕ2,…,ϕn⊨ψ\phi_1, \phi_2, \ldots, \phi_n \vDash \psiϕ1​,ϕ2​,…,ϕn​⊨ψ if and only if ϕ1∧ϕ2∧⋯∧ϕn∧¬ψ\phi_1 \land \phi_2 \land \cdots \land \phi_n \land \neg\psiϕ1​∧ϕ2​∧⋯∧ϕn​∧¬ψ is not satisfiable.
  • {}⊨ϕ\{\} \vDash \phi{}⊨ϕ if and only if ϕ\phiϕ is a tautology.
  • ϕ≡ψ\phi \equiv \psiϕ≡ψ if and only if ϕ⊨ψ\phi \vDash \psiϕ⊨ψ and ψ⊨ϕ\psi \vDash \phiψ⊨ϕ.

Theorem 10.
If ϕ≡ψ\phi \equiv \psiϕ≡ψ, then

  • ϕ\phiϕ is a tautology iff ψ\psiψ is a tautology.
  • ϕ\phiϕ is a contradiction iff ψ\psiψ is a contradiction.

Theorem 11.

  • If ϕ⊨ψ\phi \models \psiϕ⊨ψ and ψ⊨χ\psi \models \chiψ⊨χ, then ϕ⊨χ\phi \models \chiϕ⊨χ.
  • If ϕ⊨ψ\phi \models \psiϕ⊨ψ and ψ≡χ\psi \equiv \chiψ≡χ, then ϕ⊨χ\phi \models \chiϕ⊨χ.
  • If ϕ≡ψ\phi \equiv \psiϕ≡ψ and ψ⊨χ\psi \models \chiψ⊨χ, then ϕ⊨χ\phi \models \chiϕ⊨χ.

想要判断一个命题是否可满足，最直接的方法就是尝试所有真值指派。比如数独的例子，需要尝试 29×9×92^{9\times 9\times 9}29×9×9 种可能性。但计算量太大，完全穷举不合理的。

Normal Forms (范式)

Definition 12. Normal Form

  • A literal is a propositional variable or its negation. (比如说 p,q,r,¬p,¬qp, q, r, \lnot p, \lnot qp,q,r,¬p,¬q 这种)
  • A conjunctive clause (合取子句) is a conjunction of literals. (例如 ¬p∧q\lnot p \land q¬p∧q，同时 rrr 这种只有一个 literal 的也算)
  • A disjunctive clause (析取子句) is a disjunction of literals.
  • A compound proposition is in disjunctive normal form (DNF, 析取范式) if it is a disjunction of conjunctive clauses. (典型的有 (p∧q)∨(¬p∧r)(p \land q)\lor (\lnot p \land r)(p∧q)∨(¬p∧r) 这种；¬p∧q\lnot p \land q¬p∧q 也算，可以看作一个合取子句)
  • A compound proposition is in conjunctive normal form (CNF, 合取范式) if it is a conjunction of disjunctive clauses.

Remark.
A conjunctive clause may have only one literal, which can be treated as a conjunction of only one literal (退化情况). A DNF proposition may have only one conjunctive clause, which can be treated as a disjunction of only one conjunctive clause (退化情况).

Theorem 13. Every compound proposition is logically equivalent to some compound proposition in disjunctive normal form.

Theorem 14. Every compound proposition is logically equivalent to some compound proposition in conjunctive normal form.
任何复合命题都有逻辑等价的合取范式或是析取范式。

假如命题被写成 DNF 的形式，那么只要找到一个真值指派，使得其任意一个子句为真，就是可满足的。

Basic observations:

  • It is easy to determine whether a DNF proposition is SAT.
  • It is hard to determine whether a generic proposition is SAT.
  • It is impossible to turn every proposition into a concise DNF expressions (if preserving logic equivalence).
    就是说把一个复合命题转换成 DNF，这个 DNF 的长度可能是指数级的。这说明走 DNF 这条路是走不通的。
  • It is still hard to determine whether a CNF proposition is SAT, but this problem is easier to analysis.
    这个时候就需要找到一个真值指派，使得 CNF 中的每个真值指派都为真。
    但是把一个复合命题转换成 CNF，这个 CNF 的长度可能是指数级的。问题依然没有得到解决。

SAT solver

现有的解决办法是，我们把一个复合命题转化为 CNF，但是不需要逻辑等价，只需要保持可满足性相同。

Generating CNF expressions

例如

(p∧q)∨¬(q∧r)⇒(p∧q↔p1)∧(q∧r↔p2)∧(¬p2↔p3)∧(p1∨p3↔p4)∧p4⇒⋯∧⋯∧⋯∧⋯∧p4\begin{array}{ll} &(p\wedge q)\vee\neg(q\wedge r)\\ \Rightarrow&(p\wedge q\leftrightarrow p_1)\wedge(q\wedge r\leftrightarrow p_2)\wedge(\neg p_2\leftrightarrow p_3)\wedge(p_1\vee p_3\leftrightarrow p_4)\wedge p_4\\ \Rightarrow&\cdots\wedge\cdots\wedge\cdots\wedge\cdots\wedge p_4 \end{array} ⇒⇒​(p∧q)∨¬(q∧r)(p∧q↔p1​)∧(q∧r↔p2​)∧(¬p2​↔p3​)∧(p1​∨p3​↔p4​)∧p4​⋯∧⋯∧⋯∧⋯∧p4​​

  • p1,p2,p3,p4p_1, p_2, p_3, p_4p1​,p2​,p3​,p4​ 都是新引入的原子命题。第一行和第二行的式子的可满足性是一样的。
  • 不过第二行目前还不是 CNF，但是也不难将其变为长度合理的合取范式，例如

p∧q↔p1≡(p∧q→p1)∧(p1→p∧q)≡(¬(p∧q)∨p1)∧(¬p1∨(p∧q))≡(¬p∨¬q∨p1)∧(¬p1∨p)∧(¬p1∨q)\begin{aligned} &p \land q \leftrightarrow p_1 \\ \equiv &(p \land q \to p_1) \land (p_1\to p \land q) \\ \equiv &(\lnot (p \land q) \lor p_1) \land (\lnot p_1 \lor (p \land q)) \\ \equiv &(\lnot p \lor \lnot q \lor p_1) \land (\lnot p_1 \lor p) \land (\lnot p_1 \lor q) \end{aligned} ≡≡≡​p∧q↔p1​(p∧q→p1​)∧(p1​→p∧q)(¬(p∧q)∨p1​)∧(¬p1​∨(p∧q))(¬p∨¬q∨p1​)∧(¬p1​∨p)∧(¬p1​∨q)​
