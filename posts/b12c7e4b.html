Reed-Solomon 码与译码问题

Reed-Solomon (RS) 码定义

RS 码的核心思想是将离散的消息向量转化为连续的多项式，利用多项式的刚性来对抗错误。其编码过程如下：

  3. 码字 (Codeword)：设 α1,…,αq\alpha_1, \dots, \alpha_qα1​,…,αq​ 为 Fq\mathbb{F}_qFq​ 中的所有元素，计算多项式在这些点上的值，得到向量：

    (Q(α1),Q(α2),…,Q(αq))(Q(\alpha_1), Q(\alpha_2), \dots, Q(\alpha_q)) (Q(α1​),Q(α2​),…,Q(αq​))

从 Message 到 Codeword 的过程即完成了 RS 编码。

译码问题设定 (Decoding Problem)

在传输过程中，码字可能会发生改变。我们面临的是在含有大量错误的情况下恢复原多项式的问题。

问题描述

    deg⁡(R)≤max⁡(deg⁡(P0),deg⁡(Q)+deg⁡(P1))≤q2+deg⁡(Q)≤q2+k<51100q\deg(R) \le \max(\deg(P_0), \deg(Q) + \deg(P_1)) \le \frac{q}{2} + \deg(Q) \le \frac{q}{2} + k < \frac{51}{100}q deg(R)≤max(deg(P0​),deg(Q)+deg(P1​))≤2q​+deg(Q)≤2q​+k<10051​q

    根的数量严格大于 R(x)R(x)R(x) 的次数。

  4. 结论：根据代数基本定理，R(x)R(x)R(x) 必须是零多项式。即：

    P0(x)+Q(x)P1(x)≡0P_0(x) + Q(x)P_1(x) \equiv 0 P0​(x)+Q(x)P1​(x)≡0

最终，我们可以通过多项式除法直接恢复出 Q(x)Q(x)Q(x)：

Q(x)=−P0(x)P1(x)Q(x) = -\frac{P_0(x)}{P_1(x)} Q(x)=−P1​(x)P0​(x)​

注：为什么最后求出来的一定是 message 里面的 QQQ？

  1. 首先 message 里面的 QQQ 存在，且确实能够满足 P(x,Q(x))P(x, Q(x))P(x,Q(x)) 存在大于等于 51100q\frac{51}{100}q10051​q 个根
  2. RS 码译码解的唯一性保证了，在 deg⁡(Q)≤k\deg(Q)\le kdeg(Q)≤k 的前提下，只有一个解
  3. 为什么不考虑 deg⁡(Q)>k\deg(Q)>kdeg(Q)>k 的情况：因为在 deg⁡(Q)≤k\deg(Q)\le kdeg(Q)≤k 的前提下已经解出来了 QQQ。且无论 deg⁡(Q)\deg(Q)deg(Q) 是多少，解的形式总为 −P0(x)/P1(x)- P_0(x) / P_1(x)−P0​(x)/P1​(x)（有点像含二极管电路的求解，假设一个通断，解出来正确就是对的）

局部可译码码 (Locally Decodable Codes) 与 Reed-Muller 码

RS 码虽然纠错能力强，但有一个“缺点”：要恢复原始信息，通常需要读取整个码字。这便是 局部可译码码 (LDC) 想要解决的问题。Reed-Muller 码通过巧妙的几何构造实现了这一点。

Reed-Muller (RM) 码的构造

RM 码是对多元多项式的编码。首先取 Fq\mathbb{F}_qFq​ 的一个子集 {0,1,…,D}\{ 0, 1, \ldots ,D \}{0,1,…,D}，在上面考虑一个多元函数

g(x1,x2,…,xn):{0,1,…,D}n→Fqg(x_1, x_2, \ldots ,x_n): \{ 0, 1, \ldots ,D \}^{n} \to \mathbb{F}_q g(x1​,x2​,…,xn​):{0,1,…,D}n→Fq​

消息的形式为 (⋯ ,g(x1,…,xn),…)(\cdots , g(x_1, \ldots ,x_n), \ldots )(⋯,g(x1​,…,xn​),…)，即将消息作为函数 ggg 的像空间。（这里可以注意到消息的长度必然为 (D+1)n(D+1)^{n}(D+1)n，如果实际情况中消息长度没法完美对上，那就做一下补零之类的预处理）

引理：低度扩展的存在唯一性 (Low Degree Extension)
设消息 g(x1,…,xn)g(x_1, \dots, x_n)g(x1​,…,xn​) 是定义域在网格 {0,1,…,D}n⊆Fqn\{0, 1, \dots, D\}^n \subseteq \mathbb{F}_q^n{0,1,…,D}n⊆Fqn​，值域在 Fq\mathbb{F}_qFq​ 上的函数。
如果 D<qD < qD<q，则存在唯一的 nnn 元多项式 P(x1,…,xn)P(x_1, \dots, x_n)P(x1​,…,xn​)，满足：

  1. 在网格点上与 ggg 一致：对于所有 x∈{0,…,D}n\mathbf{x} \in \{0, \dots, D\}^nx∈{0,…,D}n，有 P(x)=g(x)P(\mathbf{x}) = g(\mathbf{x})P(x)=g(x)。
  2. 次数限制：PPP 关于每个变量的次数 deg⁡xi(P)≤D\deg_{x_i}(P) \le Ddegxi​​(P)≤D。

编码过程：

  • 消息：网格上的函数值 ggg。
  • 多项式：找到对应的扩展多项式 P(x1,…,xn)P(x_1, \dots, x_n)P(x1​,…,xn​)。
  • 码字：将 PPP 在整个向量空间 Fqn\mathbb{F}_q^nFqn​ 上的求值作为码字。

    (…,P(x1,…,xn),… )x∈Fqn(\dots, P(x_1, \dots, x_n), \dots)_{\mathbf{x} \in \mathbb{F}_q^n} (…,P(x1​,…,xn​),…)x∈Fqn​​

局部译码算法 (Local Decoding)

假设接收到的码字为 F(x)F(\mathbf{x})F(x)，其中存在部分错误（假设错误点数 ≤24100qn\le \frac{24}{100} q^n≤10024​qn）。
目标：恢复原始多项式在特定点 x\mathbf{x}x 处的值 P(x)P(\mathbf{x})P(x)。
核心思想：将高维空间中的纠错问题，转化为一条直线上的低维 RS 码纠错问题。

步骤 1：构造随机直线
为了求 x\mathbf{x}x 点的值，我们在空间中构造一条经过 x\mathbf{x}x 的直线 LLL。

  • 随机选择一个方向向量 a∈Fqn\mathbf{a} \in \mathbb{F}_q^na∈Fqn​。
  • 定义直线 L={at+x∣t∈Fq}L = \{ \mathbf{a}t + \mathbf{x} \mid t \in \mathbb{F}_q \}L={at+x∣t∈Fq​}。
  • 注意：当 t=0t=0t=0 时，直线经过目标点 x\mathbf{x}x。

步骤 2：限制在直线上 (Restriction)
考虑多项式 PPP 在这条直线上的取值。定义单变量多项式 p′(t)p'(t)p′(t)：

p′(t)=defP(x)∣L=P(at+x)=P(a1t+x1,…,ant+xn)p'(t) \overset{\underset{\mathrm{def}}{}}{=} P(\mathbf{x})|_{L} = P(\mathbf{a}t + \mathbf{x}) = P(a_1t+x_1, \ldots ,a_nt + x_n) p′(t)=def​P(x)∣L​=P(at+x)=P(a1​t+x1​,…,an​t+xn​)

  • 性质：p′(t)p'(t)p′(t) 是关于 ttt 的单变量多项式。
  • 次数：由于 PPP 满足 deg⁡xi(P)≤D\deg_{x_i}(P)\le Ddegxi​​(P)≤D，因此合成后的 p′(t)p'(t)p′(t) 同样满足 deg⁡(p′)xi≤D\deg(p')_{x_i}\le Ddeg(p′)xi​​≤D，总次数 deg⁡(p′)≤n⋅D\deg(p') \le n \cdot Ddeg(p′)≤n⋅D。假设 n⋅D≤q100n \cdot D \le \frac{q}{100}n⋅D≤100q​。

步骤 3：转化为 RS 码问题
我们在直线上读取接收到的值。对于直线上的每个 t=α1,…,αqt = \alpha_1, \dots, \alpha_qt=α1​,…,αq​，我们得到 F(at+x)F(\mathbf{a}t + \mathbf{x})F(at+x)。

  • 这实际上构成了一个带有噪声的单变量多项式采样（即 RS 码）。
  • 由于 a\mathbf{a}a 是随机选取的，直线上的非零点 at+x\mathbf{a}t + \mathbf{x}at+x (t≠0t \ne 0t=0) 在整个空间 Fqn\mathbb{F}_q^nFqn​ 中是均匀分布的。
  • 因此，如果整个码字的错误率较低（如 ≤24%\le 24\%≤24%），那么这条直线上的错误率也大概率较低。

步骤 4：执行 Berlekamp-Welch 并求值
现在问题变成了：已知一系列点 (t,F(at+x))(t, F(\mathbf{a}t + \mathbf{x}))(t,F(at+x))，其中大部分点满足 F(… )=p′(t)F(\dots) = p'(t)F(…)=p′(t)，请恢复 p′(t)p'(t)p′(t)。

  1. 调用算法：使用 Berlekamp-Welch 算法对这些点进行纠错，恢复出单变量多项式 p′(t)p'(t)p′(t)。
  2. 求值：我们需要的是目标点 x\mathbf{x}x 的值，对应于 t=0t=0t=0。因此计算 p′(0)p'(0)p′(0)。

    P(x)=p′(0)P(\mathbf{x}) = p'(0) P(x)=p′(0)

通过这种方法，我们只需要读取一条直线上的点（数量为 qqq），而不需要读取整个空间（数量为 qnq^nqn），就成功恢复了目标点的信息。
