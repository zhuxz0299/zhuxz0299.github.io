伽罗瓦理论 (Galois Theory)

下面考虑的域扩张都是代数扩张，并且通常也都是有限、可分扩张。

回顾：伽罗瓦群 (Galois Group)

定义：伽罗瓦群 (Galois Group)

域扩张 E/FE/FE/F 的伽罗瓦群，记为 Gal(E/F)\text{Gal}(E/F)Gal(E/F)（笔记中简记为 GF(E)G_F(E)GF​(E)），是 EEE 上所有保持 FFF 中元素不变（即 FFF-自同构）的自同构 σ:E→E\sigma: E \to Eσ:E→E 组成的群。

GF(E)={σ∈Aut(E)∣∀α∈F,σ(α)=α}G_F(E) = \{ \sigma \in \text{Aut}(E) \mid \forall \alpha \in F, \sigma(\alpha) = \alpha \} GF​(E)={σ∈Aut(E)∣∀α∈F,σ(α)=α}

这等价于 σ∣F=ι\sigma|_F = \iotaσ∣F​=ι（σ\sigmaσ 在 FFF 上的限制是恒等映射）。

  • 如果 F<L<EF < L < EF<L<E 是一个中间域 (intermediate field)，
  • 那么 GL(E)G_L(E)GL​(E) 是 GF(E)G_F(E)GF​(E) 的一个子群 (GL(E)<GF(E)G_L(E) < G_F(E)GL​(E)<GF​(E))。

不动域 (Fixed Field)

定义：不动域 (Fixed Field)

设 H<GF(E)H < G_F(E)H<GF​(E) 是伽罗瓦群的一个子群。HHH 的不动域 fix(H)fix(H)fix(H) 定义为：

fix(H)={α∈E∣∀σ∈H,σ(α)=α}fix(H) = \{ \alpha \in E \mid \forall \sigma \in H, \sigma(\alpha) = \alpha \} fix(H)={α∈E∣∀σ∈H,σ(α)=α}

即 EEE 中所有被 HHH 中 每一个 自同构 σ\sigmaσ 保持不变的元素组成的集合。

证明：fix(H)fix(H)fix(H) 是 EEE 的一个子域 (subfield)。

∀α,β∈fix(H)\forall \alpha, \beta \in fix(H)∀α,β∈fix(H)，∀σ∈H\forall \sigma \in H∀σ∈H：

  1. 加法/减法： σ(α±β)=σ(α)±σ(β)=α±β\sigma(\alpha \pm \beta) = \sigma(\alpha) \pm \sigma(\beta) = \alpha \pm \betaσ(α±β)=σ(α)±σ(β)=α±β。
    所以 (α±β)∈fix(H)(\alpha \pm \beta) \in fix(H)(α±β)∈fix(H)。
  2. 乘法/除法：（假设 β≠0\beta \neq 0β=0） σ(αβ−1)=σ(α)σ(β−1)=σ(α)σ(β)−1=αβ−1\sigma(\alpha \beta^{-1}) = \sigma(\alpha) \sigma(\beta^{-1}) = \sigma(\alpha) \sigma(\beta)^{-1} = \alpha \beta^{-1}σ(αβ−1)=σ(α)σ(β−1)=σ(α)σ(β)−1=αβ−1。
    所以 (αβ−1)∈fix(H)(\alpha \beta^{-1}) \in fix(H)(αβ−1)∈fix(H)。

因此，fix(H)fix(H)fix(H) 对域运算封闭，是 EEE 的一个子域。

伽罗瓦基本对应

利用不动域以及伽罗瓦扩张的定义，可以在 E/FE/FE/F 的中间域集合 F=def{L∣F<L<E}\mathcal{F} \overset{\underset{\mathrm{def}}{}}{=}\{ L | F<L<E \}F=def​{L∣F<L<E} 与 GF(E)G_F(E)GF​(E) 的子群集合 G=def{H∣{ι}<H<GF(E)}\mathcal{G} \overset{\underset{\mathrm{def}}{}}{=} \{ H | \{ \iota \} < H < G_{F}(E) \}G=def​{H∣{ι}<H<GF​(E)} 之间构造一个对应关系。

我们现在有两个核心映射：

  1. Π\PiΠ 映射（从域到群）：
    • L↦GL(E)F→G⋅(E)GL \mapsto G_L(E) \quad \mathcal{F} \xrightarrow{G_{\cdot}(E)} \mathcal{G}L↦GL​(E)FG⋅​(E)​G
  2. Ω\OmegaΩ 映射（从群到域）：
    • H↦fix(H)G→fix(⋅)FH \mapsto fix(H) \quad \mathcal{G} \xrightarrow{fix(\cdot )} \mathcal{F}H↦fix(H)Gfix(⋅)​F

对应的基本性质

这两个映射具有反向包含 (inclusion-reversing) 的性质：

  1. 如果 F<K<L<EF < K < L < EF<K<L<E，那么 GL(E)≤GK(E)G_L(E) \le G_K(E)GL​(E)≤GK​(E)。
  2. 如果 H<K<GF(E)H < K < G_F(E)H<K<GF​(E)，那么 fix(K)≤fix(H)fix(K) \le fix(H)fix(K)≤fix(H)。

对于任意中间域 LLL 和任意子群 HHH：

  1. L⊆fix(GL(E))L \subseteq fix(G_L(E))L⊆fix(GL​(E))
    （LLL 中的元素按定义被 GL(E)G_L(E)GL​(E) 中所有自同构固定，所以 LLL 必然在 fix(GL(E))fix(G_L(E))fix(GL​(E)) 中。）
  2. H⊆Gfix(H)(E)H \subseteq G_{fix(H)}(E)H⊆Gfix(H)​(E)
    （HHH 中的自同构按定义固定 fix(H)fix(H)fix(H) 中所有元素，所以 HHH 必然在 Gfix(H)(E)G_{fix(H)}(E)Gfix(H)​(E) 中。）

伽罗瓦理论基本定理 (Fundamental Theorem of Galois Theory)

定理：伽罗瓦理论基本定理 (Fundamental Theorem of Galois Theory)

如果 E/FE/FE/F 是一个伽罗瓦扩张，那么：

  1. 上述两个包含关系 都是等号：
    • L=fix(GL(E))L = fix(G_L(E))L=fix(GL​(E))
    • H=Gfix(H)(E)H = G_{fix(H)}(E)H=Gfix(H)​(E)
  2. Π\PiΠ 和 Ω\OmegaΩ 映射是 E/FE/FE/F 的中间域集合与 GF(E)G_F(E)GF​(E) 的子群集合之间的、互为逆的、反向包含的双射。
  3. 度/指数关系：
    • [E:L]=∣GL(E)∣[E:L] = |G_L(E)|[E:L]=∣GL​(E)∣
    • [L:F]=(GF(E):GL(E))[L:F] = (G_F(E) : G_L(E))[L:F]=(GF​(E):GL​(E)) (群的指数)

注： 如果 E/FE/FE/F 不是伽罗瓦扩张，上述等号不一定成立。
例子： 设 F=QF = \mathbb{Q}F=Q，E=Q(23)E = \mathbb{Q}(\sqrt[3]{2})E=Q(32​)。
这是一个 333 次扩张，但 EEE 不是 Q\mathbb{Q}Q 上的正规扩张。EEE 中只有恒等自同构 σ=ι\sigma = \iotaσ=ι 能保持 Q\mathbb{Q}Q 不变（因为 23\sqrt[3]{2}32​ 必须被映到 EEE 中的一个根，而 EEE 中只有一个实根 23\sqrt[3]{2}32​）。
因此 GF(E)=GQ(E)={ι}G_F(E) = G_{\mathbb{Q}}(E) = \{ \iota \}GF​(E)=GQ​(E)={ι}。
那么 fix(GF(E))=fix({ι})=E=Q(23)fix(G_F(E)) = fix(\{ \iota \}) = E = \mathbb{Q}(\sqrt[3]{2})fix(GF​(E))=fix({ι})=E=Q(32​)。
但是 F=QF = \mathbb{Q}F=Q。
所以 F≠fix(GF(E))F \neq fix(G_F(E))F=fix(GF​(E))。

抽象化：伽罗瓦连接 (Galois Connection)

伽罗瓦理论中的这种双重对应关系可以被抽象出来。

伽罗瓦连接 (Galois Connection)

定义：伽罗瓦连接 (Galois Connection)

设有两个偏序集合 (P,≤)(P, \le)(P,≤) 和 (Q,≤)(Q, \le)(Q,≤)，以及两个映射 Π:P→Q\Pi: P \to QΠ:P→Q，Ω:Q→P\Omega: Q \to PΩ:Q→P
后面为方便，记 Π(p)=p∗,p∈PΩ(q)=q′,q∈Q\Pi(p) = p^{*}, p \in P \quad \Omega(q) = q', q \in QΠ(p)=p∗,p∈PΩ(q)=q′,q∈Q

我们称 (Π,Ω)(\Pi, \Omega)(Π,Ω) 构成 (P,Q)(P, Q)(P,Q) 上的伽罗瓦连接，如果它们满足：

  1. 序反转 (Order-reversing):
    • ∀p1,p2∈P,p1≤p2  ⟹  p2∗≤p1∗\forall p_1, p_2 \in P, \quad p_1 \le p_2 \implies p_2^* \le p_1^*∀p1​,p2​∈P,p1​≤p2​⟹p2∗​≤p1∗​
    • ∀q1,q2∈Q,q1≤q2  ⟹  q2′≤q1′\forall q_1, q_2 \in Q, \quad q_1 \le q_2 \implies q_2' \le q_1'∀q1​,q2​∈Q,q1​≤q2​⟹q2′​≤q1′​
  2. 扩张 (Extension):
    • ∀p∈P,p≤(p∗)′\forall p \in P, \quad p \le (p^*)'∀p∈P,p≤(p∗)′ (简记 p≤p∗′p \le p^{*\prime}p≤p∗′)
    • ∀q∈Q,q≤(q′)∗\forall q \in Q, \quad q \le (q')^*∀q∈Q,q≤(q′)∗ (简记 q≤q′∗q \le q'^*q≤q′∗)

可以看出伽罗瓦连接正好对应了前面 F\mathcal{F}F 与 G\mathcal{G}G 之间的映射的性质。

闭包运算 (Closure Operator)

定义：闭包运算 (Closure Operator)

偏序集 PPP 上的一个映射 cl:P→Pcl: P \to Pcl:P→P 称为闭包运算，如果它满足：

  1. 扩张 (Extension): ∀p∈P,p≤cl(p)\forall p \in P, \quad p \le cl(p)∀p∈P,p≤cl(p)
  2. 幂等 (Idempotent): ∀p∈P,cl(cl(p))=cl(p)\forall p \in P, \quad cl(cl(p)) = cl(p)∀p∈P,cl(cl(p))=cl(p)
  3. 单调 (Isotonic): ∀p,q∈P,p≤q  ⟹  cl(p)≤cl(q)\forall p, q \in P, \quad p \le q \implies cl(p) \le cl(q)∀p,q∈P,p≤q⟹cl(p)≤cl(q)

伽罗瓦连接与闭包运算

定理

在一个伽罗瓦连接 (Π,Ω)(\Pi, \Omega)(Π,Ω) 中，由 clP(p)=p∗′cl_P(p) = p^{*\prime}clP​(p)=p∗′ 定义的 clP:P→Pcl_P: P \to PclP​:P→P 和由 clQ(q)=q′∗cl_Q(q) = q'^*clQ​(q)=q′∗ 定义的 clQ:Q→Qcl_Q: Q \to QclQ​:Q→Q 都是闭包运算。

证明： (以 clP(p)=p∗′cl_P(p) = p^{*\prime}clP​(p)=p∗′ 为例)

  1. 扩张： p≤p∗′p \le p^{*\prime}p≤p∗′。
    (这直接由伽罗瓦连接的定义 2 得到。)
  2. 单调： 设 p≤qp \le qp≤q。
    • 应用 ∗*∗ (序反转)：q∗≤p∗q^* \le p^*q∗≤p∗。
    • 应用 ′'′ (序反转)：p∗′≤q∗′p^{*\prime} \le q^{*\prime}p∗′≤q∗′。
    • 即 clP(p)≤clP(q)cl_P(p) \le cl_P(q)clP​(p)≤clP​(q)。
  3. 幂等： 需证 clP(clP(p))=clP(p)cl_P(cl_P(p)) = cl_P(p)clP​(clP​(p))=clP​(p)，即 p∗′∗′=p∗′p^{*\prime *\prime} = p^{*\prime}p∗′∗′=p∗′。

      • (a) 对 p∈Pp \in Pp∈P 应用扩张 (2)： p≤p∗′p \le p^{*\prime}p≤p∗′。

    • (证毕。)

抽象伽罗瓦基本定理

定理：抽象伽罗瓦基本定理

伽罗瓦连接 (Π,Ω)(\Pi, \Omega)(Π,Ω) 建立了 PPP 中所有闭元的集合 cl(P)cl(P)cl(P) 与 QQQ 中所有闭元的集合 cl(Q)cl(Q)cl(Q) 之间的一个反向同构（order-reversing isomorphism）（即反向双射）。

    显然有 Π∘Ω=Ω∘Π=ι\Pi \circ \Omega = \Omega \circ \Pi = \iotaΠ∘Ω=Ω∘Π=ι

带标度的伽罗瓦连接 (Indexed Galois Connection)

我们可以将“域扩张的度”和“群的指数”也抽象进来。

定义：带标度的伽罗瓦连接

设 (P,≤)(P, \le)(P,≤) 和 (Q,≤)(Q, \le)(Q,≤) 有一个伽罗瓦连接 (Π,Ω)(\Pi, \Omega)(Π,Ω)。

  1. 对 p,q∈Pp, q \in Pp,q∈P 且 p≤qp \le qp≤q，定义 (q:p)(q:p)(q:p) 为 qqq over ppp 的度 (degree)。
  2. 对 r,s∈Qr, s \in Qr,s∈Q 且 r≤sr \le sr≤s，定义 (s:r)(s:r)(s:r) 为 sss over rrr 的指数 (index)。
  3. 塔律 (Tower Law):
    • p1≤p2≤p3∈P  ⟹  (p3:p1)=(p3:p2)⋅(p2:p1)p_1 \le p_2 \le p_3 \in P \implies (p_3:p_1) = (p_3:p_2) \cdot (p_2:p_1)p1​≤p2​≤p3​∈P⟹(p3​:p1​)=(p3​:p2​)⋅(p2​:p1​)
    • s1≤s2≤s3∈Q  ⟹  (s3:s1)=(s3:s2)⋅(s2:s1)s_1 \le s_2 \le s_3 \in Q \implies (s_3:s_1) = (s_3:s_2) \cdot (s_2:s_1)s1​≤s2​≤s3​∈Q⟹(s3​:s1​)=(s3​:s2​)⋅(s2​:s1​)
  4. 度/指数关系：(degree nonincreasing)
    • ∀p≤q∈P  ⟹  (q:p)≥(p∗:q∗)\forall p \le q \in P \implies (q:p) \ge (p^* : q^*)∀p≤q∈P⟹(q:p)≥(p∗:q∗)
    • ∀r≤s∈Q  ⟹  (s:r)≥(r′:s′)\forall r \le s \in Q \implies (s:r) \ge (r' : s')∀r≤s∈Q⟹(s:r)≥(r′:s′)
  5. 非退化： (q:p)=1  ⟹  p=q(q:p)=1 \implies p=q(q:p)=1⟹p=q。

定理

  1. 闭元上的度保持 (Degree-preserving on closed elements):
    如果 p,r∈cl(P)p, r \in cl(P)p,r∈cl(P) (都是闭元) 且 p≤rp \le rp≤r，那么 (r:p)=(p∗:r∗)(r:p) = (p^* : r^*)(r:p)=(p∗:r∗)。

  2. 闭元的有限扩张是闭的 (Finite extensions of closed elements are closed):
    如果 p∈cl(P)p \in cl(P)p∈cl(P) (是闭元)，p≤r∈Pp \le r \in Pp≤r∈P，且 (r:p)<∞(r:p) < \infty(r:p)<∞ (有限扩张)，
    那么 rrr 也是闭元 (r∈cl(P)r \in cl(P)r∈cl(P))。

证明 (1. 度保持):

  • (a) (r:p)≥(p∗:r∗)(r:p) \ge (p^* : r^*)(r:p)≥(p∗:r∗) (由定义 4)。
  • (b) p≤r  ⟹  r∗≤p∗p \le r \implies r^* \le p^*p≤r⟹r∗≤p∗。
  • p∗p^*p∗ 和 r∗r^*r∗ 都是 QQQ 中的闭元 (因为它们是 ∗*∗ 的像)。
  • 对 QQQ 中的 r∗≤p∗r^* \le p^*r∗≤p∗ 应用定义 4：
    (p∗:r∗)≥((r∗)′:(p∗)′)(p^* : r^*) \ge ( (r^*)' : (p^*)' )(p∗:r∗)≥((r∗)′:(p∗)′)
  • 因为 p,rp, rp,r 是 PPP 中的闭元， r∗′=rr^{*\prime} = rr∗′=r 且 p∗′=pp^{*\prime} = pp∗′=p。
  • 所以 (p∗:r∗)≥(r:p)(p^* : r^*) \ge (r:p)(p∗:r∗)≥(r:p)。
  • 结合 (a) 和 (b)，得 (r:p)=(p∗:r∗)(r:p) = (p^* : r^*)(r:p)=(p∗:r∗)。

证明 (2. 有限扩张是闭的):

  • p∈cl(P)p \in cl(P)p∈cl(P)，r∈Pr \in Pr∈P，p≤rp \le rp≤r 且 (r:p)<∞(r:p) < \infty(r:p)<∞。
  • 我们需要证明 cl(r)=rcl(r) = rcl(r)=r。
  • 由闭包定义 (1)， r≤cl(r)r \le cl(r)r≤cl(r)。
  • p,cl(r)p, cl(r)p,cl(r) 都是闭元 (ppp 假设是闭元, cl(r)cl(r)cl(r) 根据定义总是闭元)。
  • p≤r≤cl(r)p \le r \le cl(r)p≤r≤cl(r)。
  • 应用塔律： (cl(r):p)=(cl(r):r)⋅(r:p)(cl(r) : p) = (cl(r) : r) \cdot (r:p)(cl(r):p)=(cl(r):r)⋅(r:p)。
  • 应用 (1. 度保持) (因为 p,cl(r)p, cl(r)p,cl(r) 是闭元)： (cl(r):p)=(p∗:cl(r)∗)(cl(r) : p) = (p^* : cl(r)^*)(cl(r):p)=(p∗:cl(r)∗)。
  • cl(r)=r∗′cl(r) = r^{*\prime}cl(r)=r∗′，所以 cl(r)∗=r∗′∗cl(r)^* = r^{*\prime*}cl(r)∗=r∗′∗。
  • 我们已知 r∗′∗=r∗r^{*\prime*} = r^*r∗′∗=r∗ (在闭包运算的证明中)。
  • 所以 (cl(r):p)=(p∗:r∗)(cl(r) : p) = (p^* : r^*)(cl(r):p)=(p∗:r∗)。
  • 另一方面，由定义 (4)： (r:p)≥(p∗:r∗)(r:p) \ge (p^* : r^*)(r:p)≥(p∗:r∗)。
  • 组合起来：

    (cl(r):r)⋅(r:p)=(cl(r):p)=(p∗:r∗)≤(r:p)(cl(r) : r) \cdot (r:p) = (cl(r) : p) = (p^* : r^*) \le (r:p) (cl(r):r)⋅(r:p)=(cl(r):p)=(p∗:r∗)≤(r:p)

  • 即 (cl(r):r)⋅(r:p)≤(r:p)(cl(r) : r) \cdot (r:p) \le (r:p)(cl(r):r)⋅(r:p)≤(r:p)。
  • 因为 (r:p)(r:p)(r:p) 是有限的，我们可以约去它。
  • 得到 (cl(r):r)≤1(cl(r) : r) \le 1(cl(r):r)≤1。
  • 根据定义 (5)，这必然意味着 (cl(r):r)=1(cl(r) : r) = 1(cl(r):r)=1，且 cl(r)=rcl(r) = rcl(r)=r。
  • 因此 rrr 是闭元。

最终总结

伽罗瓦理论基本定理是上述抽象理论的一个完美实例：

  1. 假设 E/FE/FE/F 是一个伽罗瓦扩张。
  2. 这意味着 FFF 是一个闭元 (F=fix(GF(E))F = fix(G_F(E))F=fix(GF​(E)))。
  3. 假设 E/FE/FE/F 是有限扩张 ([E:F]<∞[E:F] < \infty[E:F]<∞)。
  4. 根据定理 2 (有限扩张是闭的)，FFF 的任何有限扩张（即任何中间域 LLL）也都是闭元 (L=fix(GL(E))L = fix(G_L(E))L=fix(GL​(E)))。
  5. (同时，对于群，在伽罗瓦扩张下，所有子群 HHH 也都是闭元 H=Gfix(H)(E)H = G_{fix(H)}(E)H=Gfix(H)​(E)。)
  6. 因此，在有限伽罗瓦扩张 E/FE/FE/F 中，所有中间域和所有子群都是闭元。
  7. 根据抽象伽罗瓦基本定理，这两个集合（中间域集合与子群集合）通过 L↦GL(E)L \mapsto G_L(E)L↦GL​(E) 和 H↦fix(H)H \mapsto fix(H)H↦fix(H) 建立了一个完美的反向双射。
  8. 根据定理 1 (度保持)，这个双射还保持度与指数的相等关系：
    [L:F]=(GF(E):GL(E))[L:F] = (G_F(E) : G_L(E))[L:F]=(GF​(E):GL​(E))。
